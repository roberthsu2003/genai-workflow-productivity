# 任務 13：使用 AI 工具製作短影片（教學、簡報、社群）

## 🎯 工作情境
需要製作短影片用於教學、簡報、社群媒體宣傳等，但對影片剪輯不熟悉，或沒有時間手動編輯影片。例如：教學說明短片、產品介紹影片、社群貼文影片（Instagram Reels、TikTok）、簡報動畫等。

## 🤖 使用的 AI 工具
- **ChatGPT** 或 **Claude**（腳本撰寫和影片概念規劃）
- **Gemini Veo 3**（文字或圖片生成短影片，約 8 秒）
- **Invideo AI**（從文字腳本生成影片，免費版有浮水印）
- **Simplified** 或 **Lumen5**（文字轉影片、文章轉影片）
- **VEED** 或 **Kapwing**（影片編輯、自動字幕、音效）
- **Canva**（視覺設計素材整合）

## 💻 AI 負責的工作（80%）
- 根據文字描述生成影片腳本和大綱
- 從文字 prompt 自動生成影片內容
- 選擇和搭配視覺素材（圖片、動畫、場景）
- 自動生成字幕和語音轉文字
- 匹配背景音樂和音效
- 影片格式調整和優化

## 👤 你負責的工作（20%）
- 定義影片目標和受眾
- 撰寫或提供影片主題和核心內容
- 選擇影片風格和調性（正式/輕鬆/專業）
- 審閱和微調 AI 生成的影片內容
- 確認影片長度和格式適合平台
- 最終審核和發布

## 📝 實作步驟
1. **定義影片需求**：告訴 ChatGPT 或 Claude 你的影片目標、受眾、主題、長度（建議 15-60 秒）
2. **AI 生成腳本和大綱**：請 AI 根據需求生成影片腳本，包含開場、核心內容、結尾
3. **選擇影片生成工具**（根據需求選擇）：
   - **方法 A：Gemini Veo 3**（簡單快速，約 8 秒短片）
   - **方法 B：Invideo AI 或 Simplified**（完整影片製作）
   - **方法 C：VEED 或 Kapwing**（編輯優化）
4. **整合多個工具（進階）**：使用 ChatGPT/Claude 生成腳本 + Gemini Veo/Simplified 製作影片 + Canva 製作視覺元素
5. **影片優化和調整**：檢查字幕、確認音量平衡、調整影片格式
6. **輸出和分享**：匯出影片（MP4 格式），用於簡報、社群媒體、教學平台等

## 📄 產出結果
- **完成的短影片**（15-60 秒，包含字幕和音軌）
- **影片腳本文件**（可重複使用和修改）
- **視覺素材庫**（AI 選擇的素材可保存）
