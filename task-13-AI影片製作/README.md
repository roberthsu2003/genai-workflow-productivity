# 任務 13：使用 AI 工具製作短影片（教學、簡報、社群）

## 🎯 工作情境
需要製作短影片用於教學、簡報、社群媒體宣傳等，但對影片剪輯不熟悉，或沒有時間手動編輯影片。例如：教學說明短片、產品介紹影片、社群貼文影片（Instagram Reels、TikTok）、簡報動畫等。

## 🤖 使用的 AI 工具
- **ChatGPT** 或 **Claude**（腳本撰寫和影片概念規劃）
- **Gemini Veo 3**（文字或圖片生成短影片，約 8 秒）
- **Invideo AI**（從文字腳本生成影片，免費版有浮水印）
- **Simplified** 或 **Lumen5**（文字轉影片、文章轉影片）
- **VEED** 或 **Kapwing**（影片編輯、自動字幕、音效）
- **Canva**（視覺設計素材整合）

## 💻 AI 負責的工作（80%）
- 根據文字描述生成影片腳本和大綱
- 從文字 prompt 自動生成影片內容
- 選擇和搭配視覺素材（圖片、動畫、場景）
- 自動生成字幕和語音轉文字
- 匹配背景音樂和音效
- 影片格式調整和優化

## 👤 你負責的工作（20%）
- 定義影片目標和受眾
- 撰寫或提供影片主題和核心內容
- 選擇影片風格和調性（正式/輕鬆/專業）
- 審閱和微調 AI 生成的影片內容
- 確認影片長度和格式適合平台
- 最終審核和發布

## 📝 實作步驟
1. **定義影片需求**：告訴 ChatGPT 或 Claude 你的影片目標、受眾、主題、長度（建議 15-60 秒）
2. **AI 生成腳本和大綱**：請 AI 根據需求生成影片腳本，包含開場、核心內容、結尾
3. **選擇影片生成工具**（根據需求選擇）：
   - **方法 A：Gemini Veo 3**（簡單快速，約 8 秒短片）
   - **方法 B：Invideo AI 或 Simplified**（完整影片製作）
   - **方法 C：VEED 或 Kapwing**（編輯優化）
4. **整合多個工具（進階）**：使用 ChatGPT/Claude 生成腳本 + Gemini Veo/Simplified 製作影片 + Canva 製作視覺元素
5. **影片優化和調整**：檢查字幕、確認音量平衡、調整影片格式
6. **輸出和分享**：匯出影片（MP4 格式），用於簡報、社群媒體、教學平台等

## 📄 產出結果
- **完成的短影片**（15-60 秒，包含字幕和音軌）
- **影片腳本文件**（可重複使用和修改）
- **視覺素材庫**（AI 選擇的素材可保存）

## 💡 實際應用範例

**範例 1：教學說明短片**
- **需求**：解釋一個數學概念（如「什麼是微積分」）
- **產出**：30 秒教學短片，可在課堂使用或上傳 YouTube Shorts

**範例 2：產品介紹影片**
- **需求**：為新產品製作 45 秒的宣傳影片
- **產出**：45 秒產品宣傳影片，可用於社群媒體

**範例 3：簡報動畫短片**
- **需求**：製作簡報開場的動畫短片（15 秒）
- **產出**：15 秒動畫短片，嵌入簡報中

**範例 4：社群貼文影片（Reels/Short）**
- **需求**：製作 Instagram Reels 或 TikTok 影片（60 秒）
- **產出**：60 秒社群影片，符合平台格式要求

## ⚙️ 免費 AI 影片工具比較

| 工具 | 主要功能 | 免費方案限制 | 適用場景 |
|-----|---------|------------|---------|
| **Gemini Veo 3** | 文字/圖片→影片（8 秒） | 有次數限制 | 快速生成短片片段 |
| **Invideo AI** | 文字腳本→完整影片 | 每週有限分鐘數、有浮水印 | 完整影片製作 |
| **Simplified** | 文字/文章→影片 | 有限額度、基本功能 | 文章轉影片、教學影片 |
| **VEED** | 影片編輯、字幕生成 | 基本功能、有浮水印 | 影片編輯優化 |
| **Lumen5** | 文章/腳本→影片 | 免費版功能有限 | 內容行銷影片 |

## 🎬 影片格式建議
- **社群媒體**：9:16（直式），15-60 秒，720p 或 1080p
- **簡報嵌入**：16:9（橫式），15-30 秒，1080p
- **教學平台**：16:9 或 9:16，30-60 秒，1080p
