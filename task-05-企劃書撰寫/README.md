# 任務 5：企劃書撰寫

## 🎯 工作情境
需要撰寫一份新專案企劃書，包含市場分析、執行計畫、預算規劃、風險評估等完整內容。

## 🤖 使用的 AI 工具
- **ChatGPT** 或 **Claude**（企劃書架構與內容生成）
- **Perplexity**（市場研究與競爭分析）
- **Google Docs**（協作編輯）
- **Gemini**（多文件整合）

## 💻 AI 負責的工作（80%）
- 生成企劃書標準架構
- 撰寫各章節內容（市場分析、執行計畫、預算等）
- 提供數據和案例參考
- 格式化和排版
- 生成執行時間表（甘特圖建議）

## 👤 你負責的工作（20%）
- 定義專案目標和範圍
- 提供內部資源和限制條件
- 確認市場分析的合理性
- 決策預算分配和優先順序
- 最終審核並取得主管簽核

## 📝 實作步驟
1. **專案定義**：告訴 AI 專案目標、背景、預算範圍
2. **生成架構**：AI 產生完整的企劃書大綱（10-15 個章節）
3. **市場分析**：使用 Perplexity 查詢市場數據，請 AI 整理成分析段落
4. **內容撰寫**：逐章請 AI 撰寫，並根據你的回饋調整
5. **預算規劃**：提供成本項目，AI 協助計算和分配
6. **風險評估**：請 AI 列出可能的風險點和應對方案
7. **整合完善**：在 Google Docs 中整合所有內容，與團隊協作審核

## 📄 產出結果
- 完整的企劃書文件（Google Docs，約 20-30 頁）
- 執行時間表（可在 Google Sheets 製作甘特圖）
- 預算明細表（Excel/Sheets 格式）
