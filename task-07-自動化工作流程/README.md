# 任務 7：自動化工作流程（表單 → AI 分析 → 報告）

## 🎯 工作情境
每週收到客戶回饋表單，需要自動分析回饋內容、產生摘要報告，並發送給相關部門。

## 🤖 使用的 AI 工具
- **Google Forms**（表單收集）
- **Google Sheets**（資料儲存）
- **ChatGPT API** 或 **Claude API**（批量分析，需程式串接）
- **Google Apps Script**（自動化腳本）
- **Gmail**（自動發送）

## 💻 AI 負責的工作（80%）
- 自動讀取新提交的表單資料
- 分析回饋內容（正面/負面/中性）
- 提取關鍵意見和建議
- 生成每週摘要報告
- 自動分類並發送給對應部門

## 👤 你負責的工作（20%）
- 設計表單問題
- 設定自動化觸發條件
- 檢視 AI 分析結果是否合理
- 定義報告接收者
- 定期檢查自動化是否正常運作

## 📝 實作步驟
1. **設計表單**：在 Google Forms 建立客戶回饋表單
2. **建立自動化腳本**：使用 Google Apps Script，當新表單提交時執行分析
3. **設定觸發器**：每週自動執行分析腳本
4. **自動分發**：根據分析結果，自動發送報告給相關部門
5. **監控優化**：定期檢查自動化流程，調整分析邏輯

> **注意**：完全自動化需要 API 串接（可能需要付費）。免費方案可改為「半自動」：AI 分析部分手動執行，但仍大幅減少工作量。

## 📄 產出結果
- 自動化的表單收集系統
- 每週自動生成的回饋分析報告
- 自動通知流程（郵件/通知）
