## 📌 快速搜尋 YouTube 的建議步驟

### 給老師的搜尋建議

1. **第一步：確定搜尋目標**
   - 想要找 Google Apps Script 的基本教學？
   - 想要找 API 整合的教學？
   - 想要找自動化流程設計的教學？

2. **第二步：組合關鍵字**
   - 中文教學：使用繁體中文關鍵字
   - 英文教學：使用英文關鍵字可以找到更多專業內容
   - 加上年份：確保找到最新的教學影片

3. **第三步：篩選結果**
   - 觀看次數：選擇觀看次數較多的影片（通常品質較好）
   - 上傳日期：選擇最近一年內的影片（工具可能會更新）
   - 影片長度：10-30 分鐘的影片適合課堂播放

4. **第四步：預先觀看**
   - 在推薦給學生前，先完整觀看一遍
   - 確認內容是否符合教學需求
   - 注意是否有廣告或不適合的內容

### 給學生的搜尋建議

1. **自主學習時可以搜尋：**
   - "Google Apps Script 入門教學"
   - "如何建立自動化工作流程"
   - "API 整合教學"

2. **遇到問題時可以搜尋：**
   - 具體的問題描述，例如："Google Apps Script 觸發器設定"
   - 加上"教學"或"tutorial"關鍵字

## ⚠️ 使用 YouTube 影片的注意事項

1. **版權問題**
   - 影片為他人製作，使用時請遵守版權規定

   - YouTube 影片可能會被刪除或設為私密
   - 建議建立播放清單並定期檢查
   - 準備備用影片或替代方案

   - 確保課堂有穩定的網路連線
   - 可以事先下載影片（若授權允許）
