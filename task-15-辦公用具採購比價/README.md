# 任務 15：辦公用具採購比價

## 🎯 工作情境
需要採購辦公用具（如電腦螢幕、辦公椅、文具等），但需要比較不同供應商的價格、規格、保固和服務，手動搜尋和比價非常耗時。例如：多品項採購、預算範圍內找最佳性價比、快速交貨需求等。

## 🤖 使用的 AI 工具
- **Perplexity**（AI 瀏覽器，免費版可用，Pro 版功能更完整）
- **Manus AI 瀏覽器**（AI 驅動的瀏覽器，可自動搜尋和比較）
- **Google Sheets** 或 **Excel**（建立比價表格）
- **Google Docs**（採購提案和報告）

## 💻 AI 負責的工作（80%）
- 自動搜尋產品價格和規格資訊
- 比較多個供應商和平台的價格
- 搜尋產品評價、品牌比較、使用者反饋
- 提取關鍵規格參數（尺寸、重量、功能等）
- 比較保固、運費、交貨時間等附加條件
- 整理多品項的比價資料
- 提供推薦方案（價格最低、性價比最高、快速交貨等）

## 👤 你負責的工作（20%）
- 定義採購品項和需求規格（品項、數量、預算範圍、必備功能）
- 決定採購優先順序和限制條件（價格優先、品質優先、交貨時間優先）
- 審閱 AI 搜尋和整理的資訊準確性
- 選擇最終供應商和確認訂單細節
- 製作採購提案並提交給主管或採購部門

## 📝 實作步驟
1. **定義採購需求**：列出品項、數量、規格和預算範圍
2. **AI 搜尋比價**：使用 Perplexity 搜尋產品價格和規格資訊
3. **建立比價表格**：使用 Google Sheets 或 Excel 整理比價資料
4. **生成採購提案**：使用 AI 工具整理成完整的採購提案
5. **審閱和確認**：檢查價格準確性，選擇最終方案
6. **製作最終提案**：整理成正式文件並提交審核

## 📄 產出結果
- 完整的比價報告（包含各品項的價格、規格、供應商比較）
- 多方案比較表（價格優先、性價比、品質優先）
- 採購提案文件（包含推薦方案和預算明細）
- 資料來源引用清單（方便後續查證和確認）
