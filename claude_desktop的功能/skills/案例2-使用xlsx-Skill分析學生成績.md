# 案例 2：使用 xlsx Skill 分析學生成績

本案例展示如何使用現有的 **xlsx skill**，讓 Claude 依照最佳實踐進行數據分析，並生成專業的成績分析報告。

---

## 📋 假設情境：「分析學期成績並生成報告」

### 背景設定

**使用者：** 王老師（程式設計教師）
**使用工具：** Claude Desktop + xlsx Skill
**需求：** 
- 需要分析「Python 基礎程式設計」課程的學期成績
- 成績資料包含：平時成績、作業成績、期中成績、期末成績
- 希望生成包含統計分析、圖表、趨勢分析的完整報告

**成績資料：**
- 學生人數：30 人
- 成績項目：平時成績（30%）、作業成績（20%）、期中成績（20%）、期末成績（30%）
- 需要分析：平均分數、及格率、成績分布、各項成績關聯性

---

## 🎯 xlsx Skill 的實際應用

### 未使用 Skill 的情況

**你：** 「幫我分析這份成績資料」

**Claude 回應：** 
會進行基本分析，但可能無法生成專業的 Excel 報告，缺少圖表和統計分析。

---

### 使用 xlsx Skill 後的情況

**你：** 「請使用 xlsx skill 幫我分析這份成績資料」

**Claude 回應：**
會先讀取 `xlsx/SKILL.md`，依照最佳實踐進行數據分析，包含：
- 專業的數據分析
- 統計圖表和視覺化
- 完整的分析報告
- Excel 格式的輸出

---

## 💡 實際操作步驟

### 第一步：確認前置條件和啟用 Skills

#### 前置條件檢查

使用 Skills 功能需要滿足以下條件：

1. **訂閱方案**
   - 需要 **Pro、Max、Team 或 Enterprise** 訂閱方案
   - 免費版不支援 Skills 功能

2. **啟用必要功能**
   - **Code Execution**（程式碼執行）：必須開啟
   - **File Creation**（檔案創建）：必須開啟
   - **Skills**：必須開啟

#### 設定步驟

1. **開啟 Claude Desktop 設定**
   - 點擊左下角的「設定」圖示
   - 選擇 `Capabilities`

2. **啟用必要功能**
   - 確認 **Code Execution** 已開啟
   - 確認 **File Creation** 已開啟
   - 確認 **Skills** 已開啟

3. **驗證功能**

   在 Claude Desktop 中測試：

   ```
   請告訴我目前有哪些可用的 skills？
   ```

   如果 Claude 能夠列出 skills（包含 xlsx），表示功能正常。

---

### 第二步：準備成績資料

#### 資料格式範例

您可以提供以下格式的成績資料：

```
學生姓名,平時成績,作業成績,期中成績,期末成績
張三,85,90,88,92
李四,78,85,80,88
王五,92,88,90,95
...
```

或者直接提供 Excel 檔案。

---

### 第三步：使用 xlsx Skill 分析資料

#### **對話範例 1：基本成績分析**

**你：**
```
請使用 xlsx skill 幫我分析這份學生成績資料。

成績資料：
[貼上成績資料或上傳 Excel 檔案]

請進行以下分析：
1. 計算每位學生的總成績（平時30% + 作業20% + 期中20% + 期末30%）
2. 計算全班平均分數
3. 計算及格率（60分以上）
4. 生成成績分布圖表
5. 分析各項成績的關聯性
6. 找出需要特別關注的學生（成績較低或進步明顯）

請生成一份完整的 Excel 分析報告，包含數據表格和圖表。
```

**Claude 的回應：**
會：
1. 讀取 `xlsx/SKILL.md` 了解最佳實踐
2. 進行數據分析
3. 生成統計圖表
4. 建立 Excel 報告
5. 在 Artifacts 面板中顯示結果

**產出結果：**
- 完整的成績分析表格
- 統計圖表（長條圖、圓餅圖、散點圖等）
- 數據視覺化
- 分析報告摘要

---

#### **對話範例 2：進階數據分析**

**你：**
```
請使用 xlsx skill 幫我進行更深入的成績分析。

需求：
1. 分析各項成績的標準差和變異係數
2. 比較不同成績區間的人數分布
3. 分析平時成績與期末成績的相關性
4. 預測可能不及格的學生
5. 生成多個圖表：
   - 成績分布直方圖
   - 各項成績比較圖
   - 成績趨勢圖

請使用專業的數據分析方法和視覺化技巧。
```

**Claude 的回應：**
會：
1. 進行進階統計分析
2. 計算相關性係數
3. 生成多種圖表
4. 提供預測和建議

---

#### **對話範例 3：生成成績報告**

**你：**
```
請使用 xlsx skill 幫我生成一份完整的成績分析報告。

報告需要包含：
1. 封面頁（課程名稱、學期、分析日期）
2. 執行摘要（關鍵發現）
3. 詳細數據分析
4. 圖表和視覺化
5. 結論與建議

請使用專業的報告格式，適合向系主任或教務處報告。
```

**Claude 的回應：**
會：
1. 建立結構化的報告
2. 包含所有分析結果
3. 使用專業的報告格式
4. 提供結論和建議

---

## 📝 完整操作流程範例

以下是一個完整的對話流程，您可以複製貼上使用：

```
請使用 xlsx skill 幫我分析學生成績，需求如下：

【成績資料】
[貼上成績資料或上傳 Excel 檔案]

【分析需求】
1. 計算總成績
   - 平時成績：30%
   - 作業成績：20%
   - 期中成績：20%
   - 期末成績：30%

2. 統計分析
   - 全班平均分數
   - 及格率（60分以上）
   - 最高分和最低分
   - 標準差

3. 成績分布
   - 各分數區間的人數
   - 成績分布圖表

4. 關聯性分析
   - 各項成績之間的相關性
   - 找出影響總成績的關鍵因素

5. 特殊關注
   - 找出成績較低的學生
   - 找出進步明顯的學生

【輸出要求】
- 生成 Excel 檔案
- 包含數據表格
- 包含統計圖表
- 包含分析摘要

請依照 xlsx skill 的最佳實踐來進行分析。
```

## ⚠️ 注意事項

### 使用 Skills 的技巧

1. **明確指定使用 Skill**
   - 在對話中明確提到「使用 xlsx skill」
   - 這樣 Claude 才會主動讀取 skill 文件

2. **提供清晰的資料格式**
   - 確保資料格式正確
   - 使用標準的 CSV 或 Excel 格式

3. **指定分析需求**
   - 明確說明需要哪些分析
   - 指定圖表類型和要求

### 常見問題

**Q：Claude 沒有使用 skill 怎麼辦？**

A：請明確在對話中提到「使用 xlsx skill」，例如：
```
請使用 xlsx skill 幫我分析這份資料
```

**Q：可以分析其他類型的資料嗎？**

A：可以！xlsx skill 可以用於各種數據分析，例如：
- 銷售數據分析
- 問卷調查分析
- 實驗數據分析

**Q：可以生成哪些類型的圖表？**

A：xlsx skill 可以生成多種圖表，例如：
- 長條圖
- 圓餅圖
- 散點圖
- 折線圖
- 直方圖

**Q：如何確認 Claude 使用了 skill？**

A：Claude 通常會在回應中提到它參考了 skill，或者您可以直接詢問：
```
請告訴我你使用了哪些 skills 來進行分析？
```
