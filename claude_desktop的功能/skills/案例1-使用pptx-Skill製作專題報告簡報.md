# 案例 1：使用 pptx Skill 製作專題報告簡報

本案例展示如何使用現有的 **pptx skill**，讓 Claude 依照最佳實踐製作專業的專題報告簡報。

---

## 📋 假設情境：「製作期末專題報告簡報」

### 背景設定

**使用者：** 小華（大學生）
**使用工具：** Claude Desktop + pptx Skill
**需求：** 
- 需要製作「AI 在教育應用」的期末專題報告簡報
- 簡報需要包含：研究背景、研究方法、研究結果、結論與建議
- 希望簡報格式專業、結構清晰，符合學術報告標準

**專題資訊：**
- 專題名稱：AI 在教育應用的現況與未來發展
- 報告時間：15 分鐘
- 目標對象：教授和同學
- 簡報頁數：約 15-20 頁

---

## 🎯 pptx Skill 的實際應用

### 未使用 Skill 的情況

**你：** 「幫我製作一份專題報告簡報」

**Claude 回應：** 
會製作一份簡報，但格式可能不夠專業，結構可能不夠清晰，不符合學術報告標準。

---

### 使用 pptx Skill 後的情況

**你：** 「請使用 pptx skill 幫我製作一份專題報告簡報」

**Claude 回應：**
會先讀取 `pptx/SKILL.md`，依照最佳實踐製作符合專業標準的簡報，包含：
- 專業的簡報設計
- 清晰的結構和排版
- 適當的視覺元素
- 符合學術報告的內容組織

---

## 💡 實際操作步驟

### 第一步：確認前置條件和啟用 Skills

#### 前置條件檢查

使用 Skills 功能需要滿足以下條件：

1. **訂閱方案**
   - 需要 **Pro、Max、Team 或 Enterprise** 訂閱方案
   - 免費版不支援 Skills 功能

2. **啟用必要功能**
   - **Code Execution**（程式碼執行）：必須開啟
   - **File Creation**（檔案創建）：必須開啟
   - **Skills**：必須開啟

#### 設定步驟

1. **開啟 Claude Desktop 設定**
   - 點擊左下角的「設定」圖示
   - 選擇 `Capabilities`

2. **啟用必要功能**
   - 確認 **Code Execution** 已開啟
   - 確認 **File Creation** 已開啟
   - 確認 **Skills** 已開啟

3. **驗證功能**

   在 Claude Desktop 中測試：

   ```
   請告訴我目前有哪些可用的 skills？
   ```

   如果 Claude 能夠列出 skills（包含 pptx），表示功能正常。

---

### 第二步：使用 pptx Skill 製作簡報

#### **對話範例 1：基本專題報告簡報製作**

**你：**
```
請使用 pptx skill 幫我製作一份專題報告簡報。

專題資訊：
- 專題名稱：AI 在教育應用的現況與未來發展
- 報告時間：15 分鐘
- 目標對象：教授和同學

簡報需要包含：
1. 封面頁（專題名稱、學生姓名、指導教授、日期）
2. 目錄
3. 研究背景與動機
4. 研究目的與問題
5. 文獻回顧（至少 3 個重點）
6. 研究方法
7. 研究結果（包含圖表和數據）
8. 討論與分析
9. 結論與建議
10. 參考資料
11. 謝謝聆聽頁

請使用專業的學術簡報設計，包含適當的視覺元素。
```

**Claude 的回應：**
會：
1. 讀取 `pptx/SKILL.md` 了解最佳實踐
2. 依照專業標準製作簡報
3. 使用適當的設計和排版
4. 在 Artifacts 面板中顯示簡報

**產出結果：**
- 專業的學術簡報設計
- 清晰的結構和層次
- 適當的視覺元素（圖表、圖示等）
- 符合學術報告的內容組織

---

#### **對話範例 2：指定簡報風格和格式**

**你：**
```
請使用 pptx skill 幫我製作一份專題報告簡報，需求如下：

專題主題：社群媒體對大學生學習行為的影響
報告時間：12 分鐘

簡報要求：
1. 使用簡潔專業的設計風格（學術風格）
2. 每頁內容不要太多，保持清晰易讀
3. 數據和圖表要清楚標示來源
4. 使用圖表或流程圖說明研究架構
5. 包含問答時間提醒頁

請依照 pptx skill 的最佳實踐來製作。
```

**Claude 的回應：**
會：
1. 依照 pptx skill 的規範製作簡報
2. 使用簡潔專業的學術風格
3. 適當分配每頁內容
4. 加入數據圖表（含來源標示）
5. 使用視覺元素輔助說明

---

#### **對話範例 3：修改現有簡報**

**你：**
```
我已經有一份專題報告簡報了，請使用 pptx skill 幫我優化這份簡報。

請檢查以下問題：
1. 簡報結構是否清晰
2. 視覺設計是否專業
3. 內容組織是否合理
4. 是否符合學術報告標準

如果有需要改進的地方，請幫我修改。
```

**Claude 的回應：**
會：
1. 讀取您提供的簡報
2. 依照 pptx skill 的最佳實踐檢查
3. 提出改進建議
4. 協助修改簡報

## 📝 完整操作流程範例

以下是一個完整的對話流程，您可以複製貼上使用：

```

【專題資訊】

   - 日期

3. 研究背景與動機
   - 說明研究背景
   - 研究動機

   - 圖表說明
   - 關鍵發現

   - 結果分析

    - 列出參考文獻

【設計要求】
```

## ⚠️ 注意事項

### 使用 Skills 的技巧

1. **明確指定使用 Skill**
   - 在對話中明確提到「使用 pptx skill」
   - 這樣 Claude 才會主動讀取 skill 文件

2. **提供詳細需求**
   - 越詳細的需求，產出的簡報品質越好
   - 包含內容、格式、風格等要求

3. **檢查產出結果**
   - 在 Artifacts 面板中查看簡報
   - 可以要求 Claude 修改不滿意的地方

### 常見問題

**Q：Claude 沒有使用 skill 怎麼辦？**

A：請明確在對話中提到「使用 pptx skill」，例如：
```
請使用 pptx skill 幫我製作簡報
```

**Q：可以指定簡報的設計風格嗎？**

A：可以！在對話中說明您想要的風格，例如：
```
請使用簡潔的學術風格
請使用商務風格
請使用活潑的設計風格
```

**Q：可以修改現有的簡報嗎？**

A：可以！將簡報內容提供給 Claude，並要求使用 skill 來優化。

**Q：如何確認 Claude 使用了 skill？**

A：Claude 通常會在回應中提到它參考了 skill，或者您可以直接詢問：
```
請告訴我你使用了哪些 skills 來製作這份簡報？
```
