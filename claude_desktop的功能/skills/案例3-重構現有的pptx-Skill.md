# 案例 3：重構現有的 pptx Skill

本案例展示如何重構現有的 **pptx skill**，使其更符合您的教學需求和風格。

---

## 📋 假設情境：「自訂簡報格式以符合教學風格」

### 背景設定

**使用者：** 陳老師（程式設計教師）
**使用工具：** Claude Desktop + 自訂 pptx Skill
**需求：** 
- 現有的 pptx skill 製作的簡報很好，但不符合個人的教學風格
- 希望簡報有特定的格式要求：
  - 封面頁必須包含課程代碼和學分數
  - 每頁必須有頁碼和課程名稱
  - 程式碼範例必須使用特定的配色方案
  - 必須包含特定的圖示和視覺元素

**目標：**
- 重構現有的 pptx skill
- 加入個人化的格式要求
- 確保每次使用都能產出符合風格的簡報

---

## 🎯 重構 Skill 的實際應用

### 未重構 Skill 的情況

**你：** 「請使用 pptx skill 製作簡報」

**Claude 回應：** 
會使用標準的 pptx skill，產出的簡報雖然專業，但可能不符合您的個人風格和特定需求。

---

### 重構 Skill 後的情況

**你：** 「請使用 pptx skill 製作簡報」

**Claude 回應：**
會使用您重構後的 skill，產出的簡報完全符合您的教學風格和格式要求。

---

## 💡 實際操作步驟

### 第一步：查看現有的 Skill

1. **了解 Skill 的儲存位置**

   Skills 儲存在以下位置：
   - 系統 Skills：`/mnt/skills/` 目錄
   - 使用者自訂 Skills：`/mnt/skills/user/` 目錄

2. **查看現有的 pptx Skill**

   在 Claude Desktop 中詢問：

   ```
   請告訴我 pptx skill 的內容是什麼？它儲存在哪裡？
   ```

   Claude 會告訴您 skill 的位置和內容。

3. **了解 Skill 的結構**

   Skill 通常是一個 Markdown 文件（`SKILL.md`），包含：
   - Skill 的說明
   - 使用指南
   - 最佳實踐
   - 格式要求

---

### 第二步：重構 Skill

#### **方法一：複製並修改現有 Skill**

1. **複製現有的 Skill**

   在 Claude Desktop 中要求：

   ```
   請幫我讀取 pptx skill 的內容，並顯示完整的內容。
   ```

   將內容複製下來。

2. **建立自訂 Skill**

   在 Claude Desktop 中要求：

   ```
   請幫我在 /mnt/skills/user/ 目錄下建立一個新的 skill 資料夾，
   名稱為 "pptx-teaching"，並建立一個 SKILL.md 檔案。
   ```

3. **修改 Skill 內容**

   將複製的內容貼上，並加入您的個人化要求。

---

#### **對話範例 1：查看現有 Skill 內容**

**你：**
```
請幫我讀取 pptx skill 的完整內容，並告訴我：
1. Skill 儲存在哪裡？
2. Skill 的主要內容是什麼？
3. 有哪些格式要求和最佳實踐？
```

**Claude 的回應：**
會：
1. 讀取 pptx skill 的內容
2. 顯示 skill 的位置和結構
3. 列出主要的格式要求和最佳實踐

---

#### **對話範例 2：建立重構後的 Skill**

**你：**
```
請幫我建立一個重構後的 pptx skill，名稱為 "pptx-teaching"。

這個 skill 要基於現有的 pptx skill，但加入以下個人化要求：

【封面頁要求】
- 必須包含課程代碼（例如：CS101）
- 必須包含學分數（例如：3 學分）
- 必須包含授課教師姓名和聯絡方式

【每頁格式要求】
- 每頁必須有頁碼（格式：第 X 頁 / 共 Y 頁）
- 每頁頁腳必須包含課程名稱
- 使用特定的配色方案（主色：深藍色 #1E3A8A，輔色：淺藍色 #DBEAFE）

【程式碼範例要求】
- 程式碼必須使用特定的配色方案
- 關鍵字：藍色
- 字串：綠色
- 註解：灰色

【視覺元素要求】
- 必須使用特定的圖示風格（簡約風格）
- 圖表必須使用特定的配色方案
- 必須包含課程 Logo（如果有的話）

請建立這個 skill 的完整內容。
```

**Claude 的回應：**
會：
1. 讀取現有的 pptx skill
2. 加入您的個人化要求
3. 建立新的 skill 文件
4. 儲存在 `/mnt/skills/user/pptx-teaching/` 目錄

---

#### **對話範例 3：測試重構後的 Skill**

**你：**
```
請使用我剛才建立的 "pptx-teaching" skill 來製作一份簡報。

課程資訊：
- 課程名稱：Python 基礎程式設計
- 課程代碼：CS101
- 學分數：3 學分
- 授課教師：陳老師
- 週次：第 1 週
- 主題：Python 環境設定

請確認簡報是否符合 skill 中定義的所有格式要求。
```

**Claude 的回應：**
會：
1. 讀取您重構後的 skill
2. 依照 skill 的要求製作簡報
3. 確認所有格式要求都已符合

---

## 📝 完整操作流程範例

以下是一個完整的對話流程，您可以複製貼上使用：

```
請幫我重構 pptx skill，建立一個名為 "pptx-teaching" 的自訂 skill。

【步驟 1：查看現有 Skill】
請先讀取現有的 pptx skill 內容，並告訴我它的結構和主要要求。

【步驟 2：建立重構後的 Skill】
基於現有的 pptx skill，加入以下個人化要求：

[請在這裡列出您的個人化要求，例如：]
- 封面頁格式要求
- 每頁格式要求
- 配色方案
- 視覺元素要求
- 其他特定需求

【步驟 3：儲存 Skill】
請將重構後的 skill 儲存在 /mnt/skills/user/pptx-teaching/ 目錄下。

【步驟 4：測試 Skill】
請使用新建立的 skill 製作一份測試簡報，確認所有要求都已符合。
```

## ⚠️ 注意事項

### 重構 Skill 的技巧

1. **保留原有優點**
   - 重構時保留現有 skill 的優點和最佳實踐
   - 只修改不符合需求的部分

2. **明確的格式要求**
   - 在 skill 中明確定義所有格式要求
   - 使用具體的範例說明

3. **測試和迭代**
   - 建立 skill 後要測試
   - 根據測試結果調整 skill 內容

### 常見問題

**Q：如何查看現有的 skill 內容？**

A：在 Claude Desktop 中詢問：
```
請告訴我 pptx skill 的內容是什麼？
```

**Q：可以修改系統的 skill 嗎？**

A：不建議直接修改系統的 skill。建議建立自訂的 skill 在 `/mnt/skills/user/` 目錄下。

**Q：如何確認 skill 是否生效？**

A：使用 skill 製作簡報後，檢查是否符合 skill 中定義的要求。

**Q：可以有多個版本的 skill 嗎？**

A：可以！您可以在 `/mnt/skills/user/` 目錄下建立多個不同版本的 skill。
