# Connectors（連接器）實際操作指南

**Connectors（連接器）** 是一個讓 Claude 能夠連接到外部工具和數據源的功能系統。

## 核心概念

- **連接器就像「橋樑」**：讓 Claude 可以直接存取您的工具和數據
- **基於 MCP 標準**：所有連接器都使用 **MCP（Model Context Protocol）** 這個開放標準
- **提供更精準的回應**：Claude 能直接存取實際資料，而不是只給範本

## 主要功能

### 連接器目錄
- 這是一個精選的工具集合，您可以一鍵瀏覽和連接
- 所有連接器都基於 **MCP（Model Context Protocol）** 建立，這是 Anthropic 開發的開放標準

### 兩大類型連接器

#### 1. 網頁連接器（遠端服務）
- **服務範例**：Notion、Canva、Stripe、Linear、Slack 等
- **適用對象**：Claude 網頁版、桌面版和手機版
- **訂閱需求**：需要付費方案（Pro、Max、Team、Enterprise）

#### 2. 桌面擴展（本地應用程式）
- **服務範例**：Figma、Socket、Prisma 等
- **適用對象**：所有 Claude Desktop 用戶（包含免費用戶）
- **訂閱需求**：免費版即可使用

## 實際應用價值

透過連接器，Claude 可以：
- **直接存取您的工具和數據**：不再需要手動複製貼上資料
- **提供更相關和具體的回應**：基於實際資料產生內容，而非範本
- **在您的工具中直接協作**：與您的工作流程無縫整合

### 實際範例對比

**未連接連接器時：**
- 您問：「寫發布說明」
- Claude 回應：給您一個通用的發布說明範本

**連接 Linear 後：**
- 您問：「從 Linear 撰寫最新 sprint 的發布說明」
- Claude 回應：自動抓取 Linear 中的實際工單資料，生成具體且準確的發布說明

## 如何使用

### 基本操作步驟

#### 方法一：透過介面設定

1. **開啟連接器選單**
   - 點擊 Claude 介面左下角的「搜尋和工具」按鈕
   - 或前往「設定 > 連接器」

2. **瀏覽連接器目錄**
   - 瀏覽可用的連接器清單
   - 每個連接器都有說明，顯示其功能和適用對象

3. **選擇要連接的工具**
   - 點擊您想要連接的工具
   - 查看連接後的可用功能說明

4. **授權連接**
   - 點擊「連接」按鈕
   - 系統會導向該工具的 OAuth 認證頁面
   - 登入並授權 Claude 存取您的帳號
   - 授權完成後，連接器即可使用

#### 方法二：在對話中直接使用

連接後，您可以在對話中直接要求 Claude 使用連接器：

```
請從 Linear 取得我這個月的所有工單，並整理成進度報告
```

```
幫我從 Notion 的專案文件中找出關於使用者研究的章節
```

## 進階功能

### 自訂連接器

如果您使用的服務不在連接器目錄中，您可以建立**自訂連接器**：

- **適用方案**：Pro、Max、Team、Enterprise
- **需要技術背景**：需要了解 MCP 協議和服務的 API
- **設定位置**：透過 Claude Desktop 設定檔進行配置

### 管理已連接的連接器

1. 前往「設定 > 連接器」
2. 查看已連接的連接器清單
3. 可以隨時中斷連接或重新授權

## 注意事項

### 訂閱限制
- **網頁連接器**：需要付費方案
- **桌面擴展**：免費版可用
- 確認您的訂閱方案支援哪些連接器

### 安全考量
- OAuth 授權是安全的標準認證方式
- 您可以隨時在工具的設定中撤銷 Claude 的存取權限
- 建議定期檢查已連接的服務和權限

### 隱私保護
- Claude 只會存取您授權的資料
- 所有資料傳輸都經過加密
- 您可以隨時中斷連接

## 常見問題

**Q：連接器會不會洩露我的資料？**
A：連接器使用標準的 OAuth 認證，只會存取您明確授權的資料。您可以隨時在服務的設定中撤銷存取權限。

**Q：免費版可以使用哪些連接器？**
A：免費版可以使用「桌面擴展」類型的連接器（如 Figma、Prisma）。「網頁連接器」需要付費方案。

**Q：如何確認連接器是否正常工作？**
A：嘗試在對話中要求 Claude 使用連接器功能，例如「請從 Linear 列出我的工單」。如果連接正常，Claude 會存取實際資料並回應。

**Q：可以同時連接多個服務嗎？**
A：可以！您可以同時連接多個連接器，Claude 可以根據對話需求選擇使用哪個連接器。
