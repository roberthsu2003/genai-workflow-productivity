## 第一週：購買決策模型

### 核心概念

**購買決策過程（五階段）：**
1. **需求認知**：消費者意識到需要某個產品或服務
2. **資訊搜尋**：收集相關資訊
3. **方案評估**：比較不同選項
4. **購買決策**：選擇並購買
5. **購後評估**：使用後的滿意度評估

### 重要理論

**AIDA 模型**
- **Attention（注意）**：吸引消費者注意
- **Interest（興趣）**：引發興趣
- **Desire（欲望）**：激發購買欲望
- **Action（行動）**：促成購買行為

### 重點範例：星巴克購買決策

**流程分析：**
1. 需求認知 → 上班族早上需要咖啡提神
2. 資訊搜尋 → 知道星巴克提供咖啡
3. 方案評估 → 比較不同咖啡店（品牌形象、環境體驗）
4. 購買決策 → 選擇星巴克
5. 購後評估 → 滿意度決定回購意願

**關鍵因素：** 品牌形象、店面環境、產品品質、價格考量

## 第三週：品牌偏好形成

### 核心概念

**品牌偏好形成的五個階段：**
1. **品牌認知**：知道品牌的存在
2. **品牌理解**：了解品牌的特色
3. **品牌態度**：對品牌產生情感連結
4. **品牌偏好**：傾向選擇特定品牌
5. **品牌忠誠度**：長期選擇並推薦

### 影響因素

1. **產品體驗**：實際使用產品的感受
2. **品牌形象**：品牌傳達的價值觀
3. **社交影響**：親友的推薦和意見
4. **媒體曝光**：廣告、社群的影響
5. **價格價值**：性價比評估

### 重點範例：Apple 品牌偏好

**形成過程：**
- 品牌認知 → 透過廣告、產品曝光
- 品牌理解 → 創新、設計、品質的認知
- 品牌態度 → 產生「科技感」、「時尚」的情感連結
- 品牌偏好 → 傾向選擇 Apple 產品
- 品牌忠誠度 → 長期使用並推薦

**成功因素：** 一致的品牌形象、優質的使用體驗、社群媒體影響、情感連結建立
