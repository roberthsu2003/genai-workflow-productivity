
## 📋 假設公司背景：「科技新創公司 TechFlow」

### 客服指南範例

**公司定位：**
- 產品：雲端協作平台 SaaS 軟體
- 目標客群：中小企業、新創團隊
- 公司文化：親切、專業、快速回應
- 核心價值：讓團隊協作更簡單高效

**客服語調：**
- ✅ 親切友善但保持專業
- ✅ 用簡單易懂的方式說明技術問題
- ✅ 主動提供解決方案，不只回答問題
- ❌ 避免使用過多技術術語
- ❌ 不使用生硬的官方口吻

**常用關鍵詞：**
友善、迅速、解決方案、協助、立即處理、最佳實踐

### 常見問題與標準回覆範例

**範例 1 (帳號問題):**
```
Q: 我忘記密碼了，怎麼辦？

A: 您好！忘記密碼沒問題，您可以這樣處理：

1. 點擊登入頁面的「忘記密碼」
2. 輸入您的註冊信箱
3. 系統會立即發送重設密碼連結到您的信箱
4. 點擊連結後即可設定新密碼

如果 5 分鐘內沒收到信件，請檢查垃圾郵件匣，或聯絡我們協助處理。

需要我幫您立即處理嗎？😊
```

**範例 2 (功能疑問):**
```
Q: 如何新增團隊成員？

A: 很高興為您說明！

新增團隊成員很簡單：
✓ 進入「設定」→「團隊成員」
✓ 點擊「邀請成員」按鈕
✓ 輸入對方的 Email，選擇權限
✓ 系統會自動發送邀請信

對方點擊邀請連結即可加入團隊，立即開始協作！

如果您在操作過程遇到任何問題，隨時告訴我，我會立即協助。✨
```

**範例 3 (技術問題):**
```
Q: 檔案上傳失敗怎麼辦？

A: 檔案上傳失敗通常是以下幾個原因，我們一步步檢查：

1. **檔案大小**：單檔需在 100MB 以內
2. **網路連線**：確認網路穩定
3. **瀏覽器**：建議使用 Chrome 或 Firefox 最新版本

您可以先嘗試：
- 重新整理頁面
- 改用 Chrome 瀏覽器
- 檢查網路連線

如果還是無法上傳，請告訴我檔案類型與大小，我會立即幫您檢查！🚀
```

## 💡 在 Claude Project 中持續優化的方式

### 第一步：建立 Project 並上傳資料
1. 在 Claude Project 中上傳：
   - 客服指南文件（語調、風格規範）
   - 常見問題與標準回覆範例（20-30 題）
   - 公司政策文件（退費、服務時間等）
   - 產品功能說明文件
   - 過往優秀客服對話案例（5-10 則）

### 第二步：持續對話優化
```
你：「客戶反應很生氣，該如何安撫？」

我：[根據客服指南生成親切且專業的安撫回覆，提供解決方案]

你：「這則回覆太正式了，可以更親切一點嗎？」

我：[調整語氣，使用更多友善的用詞和表情符號]

你：「客戶問到產品比較，請提供我們與競品的優點比較」

我：[根據產品文件生成客觀且有利的比較說明]
```

### 第三步：建立可重複使用的模板
在 Project 對話中累積：
- 不同類型問題的標準回覆模板
- 常見技術問題的故障排除步驟
- 升級、續約等商務對話範本
- 道歉與問題處理的專業用語

### 第四步：收集並優化新案例
```
你：「這是今天遇到的新問題：[問題內容]，請幫我生成專業的回覆」

我：[根據客服指南生成新回覆]

你：「這個回覆很好，請把它加入我們的標準範例庫」

我：[記錄並整理成可重複使用的範例]
```
