## 📋 假設品牌背景：「晨光咖啡 Morning Glow Coffee」

- ❌ 避免過於商業化的推銷語氣

晨光、溫暖、職人、手沖、生活儀式感、片刻寧靜

## 💡 在 Claude Project 中持續優化的方式

### 第一步：建立 Project 並設定 Custom Instructions

**設定 Custom Instructions（專案指令）：**

在 Claude Project 的設定中，輸入以下指令（用白話文說明）：

```
你是晨光咖啡的行銷文案助理。你的任務是：

1. 根據品牌指南和過往文案範例，生成符合品牌調性的文案
2. 品牌語調要溫暖親切但專業，避免過度商業化
3. 文案要貼近 25-40 歲都會上班族的生活場景
4. 使用品牌關鍵詞：晨光、溫暖、職人、手沖、生活儀式感、片刻寧靜
5. 生成的文案要可以直接使用，符合社群媒體或 EDM 的格式要求
6. 如果文案需要調整，請主動詢問具體需求（例如：語氣、長度、重點）
```

**上傳資料到 Project：**
1. 品牌指南文件
2. 過往成功文案範例 (5-10 則)
3. 目標客群分析
4. 品牌視覺規範

### 第三步：建立可重複使用的模板

在 Project 對話中持續累積並優化模板：

**你：**
```
這個一週內容規劃很好，請幫我把這個格式保存下來，
以後我可以直接要求「生成本週的職場正能量內容規劃」。
```

**Claude 回覆：**
```
好的！我會記住這個格式。以後當您說「生成本週的職場正能量內容規劃」時，
我會按照這個結構（週一啟動能量、週二職場小確幸...）來生成內容。

您也可以告訴我本週的特殊活動或促銷，我會將它們自然地融入規劃中。
```

**可累積的模板類型：**
- 不同節慶的文案範本（聖誕節、新年、情人節等）
- 促銷活動固定格式（新品上市、會員優惠、限時特價等）
- 每週固定欄位內容（週一激勵文、週五放鬆文、週三知識文等）
- 不同平台格式（Instagram、Facebook、EDM、官網等）

---

### 第四步：持續優化與調整

**你：**
```
上週的社群貼文互動率不高，請分析可能的原因，
並建議如何改進。
```

**Claude 回覆：**
```
根據品牌指南分析，可能的改進方向：

1. **增加互動元素**：多使用問句，邀請讀者留言分享
2. **視覺元素**：建議加入更多生活場景的照片
3. **時機調整**：測試不同發文時間（早晨 8-9 點 vs 下午 3-4 點）
4. **Hashtag 策略**：使用更多互動性高的 Hashtag（如 #話題標籤）

建議本週可以嘗試：
- 貼文最後加上「留言分享你的咖啡時光 ⬇️」
- 加入更多生活場景的描述，讓讀者有共鳴
- 測試「週二咖啡問答」互動類型貼文
```

## 📚 課堂教學展示流程

這樣的工作流程，你可以在課堂上這樣展示給學員：

1. **第一階段：建立 Project**
   - 示範如何創建新 Project
   - 設定 Custom Instructions（專案指令）
   - 上傳品牌資料文件

2. **第二階段：基礎對話示範**
   - 展示如何要求 Claude 生成符合品牌調性的文案
   - 示範基本提問方式

3. **第三階段：優化與調整**
   - 示範如何在同一個對話串中持續優化和調整文案
   - 展示如何要求語氣、長度、重點的調整

4. **第四階段：進階應用**
   - 示範如何規劃一週內容、生成多個選項
   - 說明如何建立可重複使用的模板

5. **第五階段：Project 的優勢說明**
   - 說明 Project 的「記憶功能」如何幫助維持一致性
   - 展示如何在不同對話中都能參考品牌指南
   - 強調「一次設定，長期使用」的效率優勢
