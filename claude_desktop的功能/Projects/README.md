## **核心概念**

- **Projects 就像「專案資料夾」**：把相關的文件、對話記錄都整理在一起
- **就像整理辦公桌**：不同專案有各自的抽屜,不會混在一起

## **步驟建議**

### **第一階段：基礎操作**
1. 開啟 Claude Desktop,點擊「Projects」
2. 創建第一個 Project,命名規則示範
3. 上傳文件的方式與注意事項
4. 寫下 Custom Instructions(用白話文示範)

### **第二階段：實作演練**
1. 讓學生選一個案例建立自己的 Project
2. 上傳 2-3 個相關文件
3. 嘗試提問並得到基於文件的回答
4. 觀察 Project 如何「記住」之前的對話

### **第三階段：進階技巧**
- 展示如何調整 Custom Instructions 改變回覆風格
- 說明何時該建立新 Project vs. 在現有 Project 新增內容
- 分享檔案管理的最佳實踐

## **重點提示**

✅ **強調實際價值**：每個功能都要連結到「省時間」、「提高品質」  
✅ **避免技術術語**：用「資料夾」取代「knowledge base」  
✅ **即時示範**：現場操作比口頭解釋更有效  
✅ **動手實作**：準備好範例文件讓他們立即練習
