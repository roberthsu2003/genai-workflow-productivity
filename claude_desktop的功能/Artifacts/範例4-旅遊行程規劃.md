# 範例 4：旅遊行程規劃（休閒）

## 目標

規劃一次完整的旅遊行程，包含景點、交通、住宿和美食推薦，學習如何整理複雜資訊並進行規劃。

## 教學重點

- 學習如何整理複雜資訊
- 培養規劃和組織能力
- 體驗 AI 輔助規劃的便利性

## 操作步驟與 Prompt 對話範例

### 步驟 1：提供基本資訊並產生行程大綱

**請複製以下 prompt 貼到 Claude 對話框（記得替換成您實際的旅遊資訊）：**

```
請幫我規劃一趟「日本關西 5 天 4 夜自由行」的完整行程，並在 Artifacts 中展示 Markdown 格式的行程表。

【旅遊資訊】
- 目的地：日本關西地區（大阪、京都、奈良）
- 天數：5 天 4 夜
- 預算：每人約 3 萬元（不含機票）
- 旅遊時間：2024 年 4 月中旬（櫻花季）
- 住宿偏好：交通方便的商務旅館或民宿
- 興趣：歷史文化、美食、購物

【行程要求】
1. 每天安排 3-4 個主要景點
2. 標示景點之間的交通方式和時間
3. 推薦附近的餐廳（午餐和晚餐）
4. 考慮體力分配，不要太緊湊
5. 包含購物和休息時間

請在 Artifacts 中產生詳細的每日行程規劃表。
```

---

### 步驟 2：補充交通資訊和預估時間

**完善交通細節：**

**Prompt：**

```
請在 Artifacts 的行程表中補充詳細的交通資訊：
1. 每個景點之間的交通方式（電車、步行、計程車等）
2. 預估交通時間
3. 交通費用（如可查詢）
4. 建議使用的交通票券（如關西周遊券、大阪周遊券）

例如：
第 1 天：大阪 → 京都
- 交通：搭乘 JR 特急電車（約 30 分鐘，費用 560 日圓）
- 建議購買：關西周遊券 3 日券

請更新行程表。
```

---

### 步驟 3：加入美食推薦

**補充餐飲資訊：**

**Prompt：**

```
請在 Artifacts 的行程表中加入每日的美食推薦：
1. 早餐地點（建議在住宿附近或便利商店）
2. 午餐餐廳（位於當天景點附近，包含推薦菜色和預估價格）
3. 晚餐餐廳（可預約的高級餐廳或當地特色小店）
4. 小吃或點心推薦（景點附近）

例如：
第 2 天午餐：京都清水寺附近
- 餐廳：奧丹豆腐料理
- 推薦：湯豆腐定食（約 3,000 日圓）
- 特色：傳統京都料理

請更新行程表，讓美食推薦與景點路線搭配。
```

---

### 步驟 4：調整每日行程安排

**優化行程節奏：**

**Prompt：**

```
請檢查並優化 Artifacts 中的行程安排，確保：
1. 每天的行程不要太緊湊，有適當的休息時間
2. 景點之間的距離合理，避免浪費時間在交通上
3. 考慮參觀時間（例如：寺廟通常需要 1-2 小時）
4. 加入備用方案（如遇雨天或景點臨時關閉）

請在行程表中標示「必訪」和「備選」景點，並加入「休息時間」提醒。
```

---

### 步驟 5：加入備用方案和注意事項

**補充實用資訊：**

**Prompt：**

```
請在 Artifacts 的行程表末尾加入「備用方案」和「注意事項」章節：

【備用方案】
- 如果第 3 天下雨，室內備選景點建議
- 如果某個餐廳客滿，替代餐廳推薦
- 如果體力不支，可以刪減的景點

【注意事項】
- 各景點的開放時間和休館日
- 需要預約的餐廳或活動（附上預約方式）
- 當地文化禮儀提醒
- 緊急聯絡資訊
- 實用 APP 推薦（交通、翻譯、美食等）

請整理成清晰的列表格式。
```

---

### 步驟 6：使用 Mermaid 繪製行程時間軸（可選）

**視覺化呈現行程：**

**Prompt：**

```
請在 Artifacts 的行程表開頭，使用 Mermaid 語法繪製一個簡化的時間軸圖表，顯示：
1. 5 天的日期
2. 每天的主要城市或區域（大阪、京都、奈良）
3. 每天的核心活動（如：第一天「抵達大阪」、第二天「大阪市區觀光」）

例如：
```mermaid
gantt
    title 關西 5 日遊行程概覽
    dateFormat YYYY-MM-DD
    section 第1天
    抵達大阪: 2024-04-15
    section 第2天
    大阪市區: 2024-04-16
    section 第3天
    京都一日遊: 2024-04-17
    ...
```

請將圖表加入行程文件中。
```

## 完整範例：一次完成行程規劃

**如果您想一次完成，可以使用這個完整的 prompt：**

```
請幫我規劃一趟完整的旅遊行程，在 Artifacts 中展示詳細的 Markdown 格式行程表。

【旅遊基本資訊】
- 目的地：[填入目的地，例如：台灣環島、韓國首爾、歐洲義大利等]
- 天數：[填入天數，例如：7 天 6 夜]
- 預算：[填入預算，例如：每人 5 萬元]
- 旅遊時間：[填入日期，例如：2024 年 6 月]
- 住宿偏好：[例如：商務旅館、特色民宿、青年旅館]
- 興趣類型：[例如：自然風景、歷史文化、美食體驗、購物血拼]

【行程規劃要求】
1. 每日行程表（包含景點、交通、用餐、住宿）
2. 景點簡介和推薦理由
3. 交通方式和時間預估
4. 餐廳推薦（含價格範圍）
5. 住宿建議區域和類型
6. 預算分配建議（住宿、餐飲、交通、門票、購物等）
7. 備用方案（雨天備案、替代景點等）
8. 實用資訊（當地交通卡、必備 APP、注意事項）

【格式要求】
- 使用清晰的日期標題
- 景點和餐廳使用列表呈現
- 交通資訊用表格或列表清楚標示
- 預算用表格呈現
- 重點資訊使用粗體強調

請產出一份詳細、實用的旅遊行程規劃表。
```

**Prompt：**

```

- 時間：週六早上出發，週日晚上返回
- 預算：每人 5,000 元

請在 Artifacts 中產生詳細行程，包含：

2. 推薦的咖啡廳和餐廳

## 進階應用

### 加入預算分配表

**Prompt：**

```
請在 Artifacts 的行程表中加入「預算分配明細表」，包含：
1. 住宿費用（每晚價格 × 天數）
2. 交通費用（國際交通 + 當地交通）
3. 餐飲費用（每日預算 × 天數）
4. 門票和活動費用
5. 購物預算
6. 其他雜費（保險、小費等）

請用表格呈現，並提供預算分配的百分比建議。
```

---

### 加入天氣資訊和穿著建議

**Prompt：**

```
請在行程表中加入「天氣資訊與穿著建議」章節：
1. 當地氣候（根據旅遊月份）
2. 每日建議穿著
3. 必備物品清單
4. 特殊天氣注意事項（如雨季、雪季）

請整合到 Artifacts 的行程文件中。
```

---

## 使用提示

1. **資訊越詳細越好**：提供越多偏好和限制，產出的行程會越貼近需求
2. **靈活調整**：在 Artifacts 中可以直接編輯和調整行程
3. **結合實際查詢**：AI 提供的資訊建議再與實際的營業時間、價格等交叉確認
4. **保留彈性**：行程不要排得太滿，保留彈性時間應對突發狀況
5. **備用方案**：規劃時就考慮備用方案，旅行更順利
