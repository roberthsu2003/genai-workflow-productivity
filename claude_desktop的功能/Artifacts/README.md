# Artifacts 實際操作指南

## 核心概念

Artifacts 將對話與作品區分開來：
- **左側聊天區**：與 Claude 的對話互動
- **右側作品區**：專注展示和編輯生成內容

就像在迷你 IDE 或文件編輯器中工作，這些內容可以獨立開啟、專心編輯，並反覆讓 Claude 協助調整。

## 主要特色

### 即時編輯與迭代

- 可以直接在 Artifacts 中修改 React 元件、HTML、Markdown、Mermaid 圖表等內容
- 修改後可以再請 Claude 根據您的調整繼續優化
- 支援多種格式：React 元件、HTML、Markdown、Mermaid 圖表等

### 清晰展示

- 上課或展示時，左側保留對話記錄，右側專注展示程式碼或圖表
- 學生不會被對話記錄干擾，更容易專注於實際內容
- 適合即時示範和互動教學

## 教學應用

### 課堂示範流程

1. 讓 Claude 先產生範例程式或圖表
2. 使用 Artifacts 直接展示「修改程式碼、重構、修復 bug」的完整過程
3. 學生可以清楚看到每個修改步驟和效果

### 活教材應用

- 將同一份 Artifacts 當作「活教材」
- 一邊修改一邊詢問 Claude，示範人機協作的工作流程
- 學生可以觀察 AI 如何根據回饋持續改進內容

## 支援的內容類型

Artifacts 不僅限於程式碼，所有「可視化的生成內容」都可以展示和編輯：

### 📄 文字文件

- 講義、教學文件、學習單、企劃稿等
- 使用 Markdown 或類似格式在 Artifacts 中編輯
- 格式清晰，易於後續轉換為其他文件格式

### 📊 圖表與架構圖

- Mermaid 流程圖、架構圖、時序圖
- 用來說明系統設計或教學流程
- 視覺化呈現，讓複雜概念更易理解

### 🌐 網頁內容

- HTML、簡單的前端畫面設計
- 不一定要具備程式概念，也可當作展示版面工具
- 適合快速原型製作和版面規劃

## 實務應用場景

### 📝 報告與簡報草稿

- 先請 Claude 產生大綱與內容
- 在 Artifacts 中細修和調整
- 完成後轉換為 Word 或投影片格式

### 📋 資料整理

- 使用文字和表格（Markdown）彙整規格、需求、會議紀錄等
- 保持與對話分開，閱讀更清楚
- 結構化呈現資訊，便於後續參考

## 範例大綱

以下是 4 個實際操作範例，涵蓋程式、工作和休閒場景。每個範例都包含詳細的操作步驟和可直接複製貼上使用的 Prompt 對話範例：

### 📌 [範例 1：建立互動式待辦清單網頁（程式）](./範例1-互動式待辦清單網頁.md)

**目標**：使用 HTML、CSS 和 JavaScript 建立一個功能完整的待辦清單應用

**教學重點**：
- 展示前端開發的完整流程
- 示範如何逐步迭代功能
- 學習程式碼結構和最佳實踐

**快速預覽**：
1. 請 Claude 產生基本的 HTML 結構和樣式
2. 在 Artifacts 中檢視和修改程式碼
3. 要求 Claude 加入新功能（如標記完成、刪除項目）
4. 即時調整樣式和互動效果
5. 示範如何除錯和優化程式碼

---

### 📌 [範例 2：會議記錄整理與摘要（工作）](./範例2-會議記錄整理與摘要.md)

**目標**：將會議討論內容整理成結構化的會議記錄

**教學重點**：
- 學習如何將雜亂資訊結構化
- 掌握專業文件格式
- 提升資訊整理能力

**快速預覽**：
1. 提供會議錄音轉文字或筆記內容給 Claude
2. 請 Claude 產生標準化的會議記錄格式
3. 在 Artifacts 中直接編輯重點事項和決議
4. 要求 Claude 補充行動項目和負責人
5. 調整格式和內容，完成最終版本

---

### 📌 [範例 3：專案企劃書撰寫（工作）](./範例3-專案企劃書撰寫.md)

**目標**：建立一份完整的專案企劃書，包含背景、目標、執行方案和預算

**教學重點**：
- 學習企劃書的標準格式
- 培養邏輯思考和結構化寫作能力
- 了解如何整合多種內容類型

**快速預覽**：
1. 說明專案需求，請 Claude 產生企劃書大綱
2. 在 Artifacts 中逐項完善內容
3. 請 Claude 補充數據分析和風險評估
4. 調整文字表達和結構邏輯
5. 加入圖表和視覺化元素（Mermaid 流程圖）

---

### 📌 [範例 4：旅遊行程規劃（休閒）](./範例4-旅遊行程規劃.md)

**目標**：規劃一次完整的旅遊行程，包含景點、交通、住宿和美食推薦

**教學重點**：
- 學習如何整理複雜資訊
- 培養規劃和組織能力
- 體驗 AI 輔助規劃的便利性

**快速預覽**：
1. 說明旅遊目的地、天數和預算限制
2. 請 Claude 產生行程規劃表（Markdown 表格）
3. 在 Artifacts 中調整每日行程安排
4. 要求 Claude 補充交通資訊和預估時間
5. 加入備用方案和注意事項
6. 可選：使用 Mermaid 繪製行程時間軸圖表

> 💡 **提示**：每個範例檔案都包含完整的 Prompt 對話範例，學生可以直接複製貼上使用，無需自己構思 Prompt。請點擊上方連結查看詳細內容。

## 教學延伸應用

除了程式課程外，Artifacts 也適合其他教學場景：

- **AI 素養課程**：展示 AI 生成的文件範例
- **專題企劃課程**：作為「範例文件／專題模板」專用區
- 任何需要展示結構化內容的課程都可以活用 Artifacts

## 重點提示

✅ **強調實際價值**：將對話與作品分開，讓展示更清晰、編輯更專注  
✅ **即時示範**：現場操作 Artifacts，讓學生看到即時編輯和迭代的過程  
✅ **活用場景**：不僅限於程式碼，各種文件、圖表、網頁都可以展示  
✅ **互動教學**：使用 Artifacts 作為「活教材」，示範人機協作的工作流程
