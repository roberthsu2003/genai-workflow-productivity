## 💡 實際操作步驟

### 第一步：安裝和設定新聞摘要 MCP Server

#### 前置準備

在開始之前，請確認您的電腦已安裝以下工具：

1. **Node.js**（如果還沒安裝）
   - 前往 [Node.js 官網](https://nodejs.org/) 下載安裝
   - 或使用套件管理器安裝（macOS 使用 Homebrew：`brew install node`）
   - 安裝後，開啟終端機輸入 `node --version` 確認安裝成功

#### 設定步驟

1. **開啟 Claude Desktop 設定**
   
   方法一（推薦）：透過 Claude Desktop 介面設定
   - 開啟 Claude Desktop 應用程式
   - 點擊左下角的「設定」圖示
   - 選擇 `Developer` → `MCP`
   - 在這裡可以直接編輯 MCP 伺服器設定
   
   方法二：直接編輯設定檔
   - **Mac**：`~/Library/Application Support/Claude/claude_desktop_config.json`
   - **Windows**：`%APPDATA%\Claude\claude_desktop_config.json`
   - 如果檔案不存在，請建立一個新的 JSON 檔案

2. **編輯設定檔內容**

   在設定檔中加入以下內容（請將 API Key 替換為您的實際 API Key）：

   ```json
   {
     "mcpServers": {
       "news": {
         "command": "npx",
         "args": [
           "-y",
           "@modelcontextprotocol/server-newsapi"
         ],
         "env": {
           "NEWS_API_KEY": "您的_API_Key_這裡"
         }
      }
    }
   }
   ```

   **⚠️ 重要提示：**
   - 您需要先註冊 [NewsAPI](https://newsapi.org/) 帳號取得免費 API Key
   - 免費版每日可取得 100 則新聞
   - 請將 `您的_API_Key_這裡` 替換為實際的 API Key

3. **如何取得 NewsAPI Key**

   - 前往 [NewsAPI 官網](https://newsapi.org/)
   - 點擊「Get API Key」註冊帳號
   - 登入後，在 Dashboard 中可以看到您的 API Key
   - 複製 API Key 並貼到設定檔中

4. **儲存並重啟 Claude Desktop**
   - 儲存設定檔
   - 完全關閉 Claude Desktop
   - 重新開啟 Claude Desktop

5. **驗證連接**

   在 Claude Desktop 中測試：

   ```
   請取得一則最新的科技新聞
   ```

   如果 Claude 能夠取得新聞，表示 MCP Server 連接成功！

## ⚠️ 注意事項

### API Key 管理

1. **API Key 安全性**
   - 請妥善保管您的 API Key，不要分享給他人
   - 設定檔應該放在安全的位置，不要上傳到公開的程式碼庫

2. **免費版限制**
   - NewsAPI 免費版每日限額 100 則新聞
   - 如果超過限制，需要等待隔天或升級方案

3. **其他新聞來源**
   - 除了 NewsAPI，還有其他 MCP Server 可以連接不同新聞來源
   - 您可以根據需求選擇合適的新聞來源

### 常見問題

**Q：設定後還是無法取得新聞？**

A：請檢查：
- API Key 是否正確輸入（沒有多餘的空格）
- JSON 格式是否正確
- 是否有重新啟動 Claude Desktop
- NewsAPI 帳號是否正常運作（可以到官網測試）

**Q：免費版 API 限制是多少？**

A：
- 每日可取得 100 則新聞
- 適用於個人學習和小型專案
- 如果需要更多，可以考慮升級方案

**Q：可以取得中文新聞嗎？**

A：可以在搜尋時指定語言，例如：
```
請搜尋「科技」相關的中文新聞
```

**Q：有其他新聞來源的 MCP Server 嗎？**

A：是的，您可以在 MCP Server 目錄中搜尋其他新聞來源。不同的新聞來源可能有不同的特色和限制。
