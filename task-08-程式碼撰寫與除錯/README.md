# 任務 8：程式碼撰寫與除錯

## 🎯 工作情境
需要開發一個功能或修改現有程式碼，但對某些語法、邏輯或框架不熟悉，需要 AI 協助生成程式碼、解釋邏輯和除錯。

## 🤖 使用的 AI 工具
- **ChatGPT**（程式碼生成與解釋）
- **Claude**（長程式碼分析）
- **GitHub Copilot**（VS Code 擴充功能，自動程式碼建議）
- **Perplexity**（技術文件查詢）

## 💻 AI 負責的工作（80%）
- 根據需求生成程式碼
- 解釋程式碼邏輯和語法
- 除錯和修正錯誤
- 重構程式碼
- 生成測試案例
- 提供程式碼優化建議

## 👤 你負責的工作（20%）
- 清楚描述功能需求
- 理解 AI 生成的程式碼邏輯
- 測試並驗證功能正確性
- 整合到現有專案
- 程式碼審查和優化
- 確保程式碼符合專案規範

## 📝 實作步驟
1. **描述需求**：清楚告訴 ChatGPT/Claude 你想要的功能和需求
2. **程式碼生成**：AI 產生程式碼，並解釋關鍵邏輯和語法
3. **除錯修正**：遇到錯誤時，將錯誤訊息貼給 AI，要求修正
4. **使用 GitHub Copilot**：在 VS Code 中，Copilot 會自動建議程式碼
5. **文件查詢**：使用 Perplexity 查詢最新技術文件和 API 說明
6. **測試驗證**：執行並測試，確保功能正常

## 📄 產出結果
- 可執行的程式碼檔案
- 程式碼註解和說明文件
- 測試案例和執行結果
- 錯誤修正記錄
