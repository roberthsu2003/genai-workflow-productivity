# 任務 8：程式碼撰寫與除錯（開發者適用）

## 🎯 工作情境
需要開發一個簡單的功能，或修改現有程式碼，但對某些語法或邏輯不熟悉。

## 🤖 使用的 AI 工具
- **ChatGPT**（程式碼生成與解釋）
- **Claude**（長程式碼分析）
- **GitHub Copilot**（免費版，VS Code 擴充功能）
- **Perplexity**（技術文件查詢）

## 💻 AI 負責的工作（80%）
- 根據需求生成程式碼
- 解釋程式碼邏輯
- 除錯和修正錯誤
- 重構程式碼
- 生成測試案例

## 👤 你負責的工作（20%）
- 清楚描述功能需求
- 理解 AI 生成的程式碼邏輯
- 測試並驗證功能正確性
- 整合到現有專案
- 程式碼審查和優化

## 📝 實作步驟
1. **描述需求**：清楚告訴 ChatGPT 你想要的功能
2. **程式碼生成**：AI 產生程式碼，並解釋關鍵邏輯
3. **除錯修正**：遇到錯誤時，將錯誤訊息貼給 AI，要求修正
4. **使用 GitHub Copilot**：在 VS Code 中，Copilot 會自動建議程式碼
5. **文件查詢**：使用 Perplexity 查詢最新技術文件
6. **測試驗證**：執行並測試，確保功能正常

## 📄 產出結果
- 可執行的程式碼
- 程式碼註解和說明文件
- 測試案例
