# 任務 1：撰寫會議紀錄與摘要

## 🎯 工作情境
每週團隊會議後，需要將 60 分鐘的會議內容整理成結構化的紀錄，並提取關鍵決議事項。

## 🤖 使用的 AI 工具
- **NotebookLM**（Google Meet 錄音/逐字稿上傳）
- **Claude** 或 **ChatGPT**（結構化整理）
- **Google Docs**（協作編輯）

## 💻 AI 負責的工作（80%）
- 自動轉錄語音成文字
- 識別並提取討論主題
- 整理發言重點
- 生成會議結構（討論內容、決議事項、行動項目）
- 格式化文件

## 👤 你負責的工作（20%）
- 確認會議錄音/逐字稿上傳
- 檢視並修正 AI 提取的關鍵點
- 確認決議事項的準確性
- 標記負責人和截止日期
- 最終審核並發送

## 📝 實作步驟
1. **上傳會議資料**：將 Google Meet 錄音或逐字稿上傳至 NotebookLM
2. **AI 自動摘要**：讓 NotebookLM 生成會議摘要
3. **結構化整理**：將摘要貼到 Claude/ChatGPT，使用 Prompt 進行結構化整理
4. **協作完善**：將結果貼到 Google Docs，與團隊成員協作確認
5. **分發通知**：確認後透過 Gmail 發送給相關人員

## 📄 產出結果
- 結構化的會議紀錄文件（Google Docs）
- 可直接用的會議摘要（5 分鐘閱讀版本）
- 行動項目清單（追蹤用）
