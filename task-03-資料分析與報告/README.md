# 任務 3：資料分析與報告撰寫

## 🎯 工作情境
每月需要分析銷售數據，產出一份包含趨勢分析、問題發現、建議方案的報告。

## 🤖 使用的 AI 工具
- **Google Sheets**（資料處理，AI 公式生成）
- **ChatGPT** 或 **Claude**（分析邏輯、報告撰寫）
- **Gemini**（多表整合分析）
- **Canva**（資訊圖表設計，Magic Design 生成視覺化圖表）
- **Google Nano Banana Pro**（圖表生成）

## 💻 AI 負責的工作（80%）
- 自動計算統計數據（平均值、成長率、趨勢）
- 生成資料視覺化建議
- 識別異常數據點
- 撰寫分析段落
- 生成報告結構

## 👤 你負責的工作（20%）
- 提供原始數據或數據來源
- 定義分析目標和關鍵指標
- 驗證 AI 發現的問題是否合理
- 決定建議方案的優先順序
- 最終報告的決策和簽核

## 📝 實作步驟
1. **數據準備**：將銷售數據整理到 Google Sheets
2. **AI 公式生成**：請 ChatGPT 生成需要的分析公式
3. **數據分析**：將數據貼給 Claude，要求分析趨勢、找出異常
4. **視覺化**：根據 AI 建議，選擇 Google Sheets、Canva 或 Nano Banana Pro 製作圖表
5. **報告撰寫**：請 AI 根據分析結果撰寫報告
6. **完善與分發**：確認數據準確性後，匯出為 PDF 或分享連結

## 📄 產出結果
- 資料分析報告（Google Docs）
- 視覺化圖表（內嵌在報告中）
- 數據異常清單（需進一步調查的項目）
