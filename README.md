## 任務 1：撰寫會議紀錄與摘要

### 🎯 工作情境
每週團隊會議後，需要將 60 分鐘的會議內容整理成結構化的紀錄，並提取關鍵決議事項。

### 🤖 使用的 AI 工具
- **NotebookLM**（Google Meet 錄音/逐字稿上傳）
- **Claude** 或 **ChatGPT**（結構化整理）
- **Google Docs**（協作編輯）

### 💻 AI 負責的工作（80%）
- 自動轉錄語音成文字
- 識別並提取討論主題
- 整理發言重點
- 生成會議結構（討論內容、決議事項、行動項目）
- 格式化文件

### 👤 你負責的工作（20%）
- 確認會議錄音/逐字稿上傳
- 檢視並修正 AI 提取的關鍵點
- 確認決議事項的準確性
- 標記負責人和截止日期
- 最終審核並發送

### 📝 實作步驟
1. **上傳會議資料**：將 Google Meet 錄音或逐字稿上傳至 NotebookLM
2. **AI 自動摘要**：讓 NotebookLM 生成會議摘要
3. **結構化整理**：將摘要貼到 Claude/ChatGPT，使用 Prompt：
   ```
   請將以下會議內容整理成結構化的會議紀錄，包含：
   - 會議資訊（時間、參與者）
   - 討論主題（標題 + 重點摘要）
   - 決議事項
   - 行動項目（包含負責人和截止日期）
   ```
4. **協作完善**：將結果貼到 Google Docs，與團隊成員協作確認
5. **分發通知**：確認後透過 Gmail 發送給相關人員

### 📄 產出結果
- 結構化的會議紀錄文件（Google Docs）
- 可直接用的會議摘要（5 分鐘閱讀版本）
- 行動項目清單（追蹤用）

## 任務 2：製作簡報（從零到完成）

### 🎯 工作情境
需要在 3 天內完成一份 20 頁的產品提案簡報，包含市場分析、產品介紹、財務預測。

### 🤖 使用的 AI 工具
- **ChatGPT** 或 **Claude**（內容大綱生成）
- **Google Slides**（簡報製作，內建 AI 輔助）
- **Canva**（視覺化簡報設計，Magic Design 生成模板）
- **Perplexity**（市場數據研究）
- **Gemini**（多模態內容整合）

### 💻 AI 負責的工作（80%）
- 根據主題生成簡報大綱
- 撰寫各頁文字內容
- 生成圖表數據建議
- 設計簡報結構和流程
- 提供視覺化建議

### 👤 你負責的工作（20%）
- 定義簡報目標和受眾
- 提供產品核心資訊
- 確認數據來源的準確性
- 調整簡報風格符合品牌
- 最終簡報前的檢查和潤飾

### 📝 實作步驟
1. **定義需求**：告訴 ChatGPT 你的簡報目標、受眾、核心訊息
2. **生成大綱**：AI 產生 20 頁的簡報架構
3. **內容生成**：逐頁請 AI 撰寫內容，並用 Perplexity 查詢市場數據
4. **簡報製作（二選一）**：
   - **方法 A：Google Slides**：使用 AI 輔助功能貼上內容
   - **方法 B：Canva**：使用 Magic Design，輸入主題和風格提示，自動生成簡報模板
5. **視覺優化**：
   - **Google Slides**：請 AI 建議圖表類型，在 Google Sheets 製作後插入
   - **Canva**：使用 Magic Design 生成視覺化圖表，或從免費素材庫選擇
6. **最終調整**：檢查邏輯流程、修正品牌色調、確認數據準確性

### 📄 產出結果
- 完整的 20 頁簡報（Google Slides）
- 簡報大綱文件（可作為會議討論基礎）
- 數據來源清單（便於後續更新）

## 任務 3：資料分析與報告撰寫

### 🎯 工作情境
每月需要分析銷售數據，產出一份包含趨勢分析、問題發現、建議方案的報告。

### 🤖 使用的 AI 工具
- **Google Sheets**（資料處理，AI 公式生成）
- **ChatGPT** 或 **Claude**（分析邏輯、報告撰寫）
- **Gemini**（多表整合分析）
- **Canva**（資訊圖表設計，Magic Design 生成視覺化圖表）
- **Google Nano Banana Pro**（圖表生成）

### 💻 AI 負責的工作（80%）
- 自動計算統計數據（平均值、成長率、趨勢）
- 生成資料視覺化建議
- 識別異常數據點
- 撰寫分析段落
- 生成報告結構

### 👤 你負責的工作（20%）
- 提供原始數據或數據來源
- 定義分析目標和關鍵指標
- 驗證 AI 發現的問題是否合理
- 決定建議方案的優先順序
- 最終報告的決策和簽核

### 📝 實作步驟
1. **數據準備**：將銷售數據整理到 Google Sheets
2. **AI 公式生成**：請 ChatGPT 生成需要的分析公式（SUM、AVERAGE、GROWTH RATE 等）
3. **數據分析**：將數據貼給 Claude，要求分析趨勢、找出異常
4. **視覺化**（三選一）：
   - **方法 A：Google Sheets**：根據 AI 建議製作圖表
   - **方法 B：Canva**：使用 Magic Design，輸入數據主題生成資訊圖表模板，手動填入數據
   - **方法 C：Nano Banana Pro**：生成資訊圖表
5. **報告撰寫**：請 AI 根據分析結果撰寫報告，包含：
   - 執行摘要
   - 數據解讀
   - 問題分析
   - 建議方案
6. **完善與分發**：確認數據準確性後，匯出為 PDF 或分享 Google Docs 連結

### 📄 產出結果
- 資料分析報告（Google Docs）
- 視覺化圖表（內嵌在報告中）
- 數據異常清單（需進一步調查的項目）

## 任務 4：郵件撰寫與回覆

### 🎯 工作情境
每天需要回覆大量客戶詢問郵件，內容包含產品諮詢、技術支援、客訴處理等不同類型。

### 🤖 使用的 AI 工具
- **Gmail 智慧撰寫**（內建 AI 建議）
- **ChatGPT** 或 **Claude**（郵件草稿生成）
- **Gemini**（多郵件批量處理）

### 💻 AI 負責的工作（80%）
- 根據郵件內容自動分類（詢問/客訴/感謝）
- 生成回覆草稿
- 調整語氣（正式/友善/專業）
- 翻譯多語言郵件
- 提取客戶郵件的關鍵資訊

### 👤 你負責的工作（20%）
- 確認郵件分類是否正確
- 檢查 AI 回覆是否符合公司政策
- 加入個人化內容（客戶名稱、具體案例）
- 決定回覆的優先順序
- 最終確認後發送

### 📝 實作步驟
1. **郵件分類**：將客戶郵件貼給 ChatGPT，要求分類並提取關鍵資訊
2. **生成回覆**：請 AI 根據郵件類型和公司模板生成回覆草稿
3. **調整語氣**：指定語氣（正式/親切），AI 調整用詞
4. **個人化**：手動加入客戶名稱和具體細節
5. **利用 Gmail 智慧撰寫**：在 Gmail 中撰寫時，AI 會自動建議完整句子
6. **批量處理**：將多封相似郵件一併處理，使用模板快速回覆

### 📄 產出結果
- 分類好的郵件清單（優先順序）
- 標準回覆模板庫（可重複使用）
- 已回覆的郵件記錄（追蹤用）

## 任務 7：自動化工作流程（表單 → AI 分析 → 報告）

### 🎯 工作情境
每週收到客戶回饋表單，需要自動分析回饋內容、產生摘要報告，並發送給相關部門。

### 🤖 使用的 AI 工具
- **Google Forms**（表單收集）
- **Google Sheets**（資料儲存）
- **ChatGPT API** 或 **Claude API**（批量分析，需程式串接）
- **Google Apps Script**（自動化腳本）
- **Gmail**（自動發送）

### 💻 AI 負責的工作（80%）
- 自動讀取新提交的表單資料
- 分析回饋內容（正面/負面/中性）
- 提取關鍵意見和建議
- 生成每週摘要報告
- 自動分類並發送給對應部門

### 👤 你負責的工作（20%）
- 設計表單問題
- 設定自動化觸發條件
- 檢視 AI 分析結果是否合理
- 定義報告接收者
- 定期檢查自動化是否正常運作

### 📝 實作步驟
1. **設計表單**：在 Google Forms 建立客戶回饋表單
2. **建立自動化腳本**：使用 Google Apps Script，當新表單提交時：
   - 讀取資料
   - 呼叫 ChatGPT/Claude API（或使用免費額度手動批次處理）
   - 分析回饋內容
   - 生成摘要
3. **設定觸發器**：每週自動執行分析腳本
4. **自動分發**：根據分析結果，自動發送報告給相關部門
5. **監控優化**：定期檢查自動化流程，調整分析邏輯

> **注意**：完全自動化需要 API 串接（可能需要付費）。免費方案可改為「半自動」：AI 分析部分手動執行，但仍大幅減少工作量。

### 📄 產出結果
- 自動化的表單收集系統
- 每週自動生成的回饋分析報告
- 自動通知流程（郵件/通知）

- **Google Drive**（如需引用檔案做為上下文）

### 💻 AI 負責的工作（80%）
- 執行重複性提示和指令
- 自動生成文字、摘要、校稿、建議
- 依照設定角色風格處理輸入內容
- 保持一致的輸出品質和風格
- 根據上下文檔案提供客製化建議

### 👤 你負責的工作（20%）
- 設定 Gem 的角色與指令（角色、語氣、任務類型）

   例如：產品競爭分析
   - 告訴 Gemini：「請從我的 Gmail 郵件、Google Drive 文件、以及 Chat 記錄中，找出最近三個月關於競爭對手的資訊，分析他們的市場策略和定價模式」
   - Gemini 會自動：
     - 搜尋相關的郵件、文件和聊天記錄
     - 分析內容並提取關鍵資訊
     - 整合多個來源的資料
     - 生成結構化的分析報告

5. **審閱和調整**：
   - 檢查 AI 生成的分析內容
   - 確認資料來源的準確性
   - 查看資料引用（Deep Research 通常會標註資料來源）
   - 微調分析重點和格式

6. **輸出和應用**：
   - 將研究報告匯出為文件（Word、PDF）
   - 或直接在 Gemini 中繼續深入討論
   - 分享給團隊成員或用於簡報

## 任務 13：使用 AI 工具製作短影片（教學、簡報、社群）

### 🎯 工作情境
需要製作短影片用於教學、簡報、社群媒體宣傳等，但對影片剪輯不熟悉，或沒有時間手動編輯影片。例如：教學說明短片、產品介紹影片、社群貼文影片（Instagram Reels、TikTok）、簡報動畫等。

### 🤖 使用的 AI 工具
- **ChatGPT** 或 **Claude**（腳本撰寫和影片概念規劃）
- **Gemini Veo 3**（文字或圖片生成短影片，約 8 秒）
- **Invideo AI**（從文字腳本生成影片，免費版有浮水印）
- **Simplified** 或 **Lumen5**（文字轉影片、文章轉影片）
- **VEED** 或 **Kapwing**（影片編輯、自動字幕、音效）
- **Canva**（視覺設計素材整合）

### 💻 AI 負責的工作（80%）
- 根據文字描述生成影片腳本和大綱
- 從文字 prompt 自動生成影片內容
- 選擇和搭配視覺素材（圖片、動畫、場景）
- 自動生成字幕和語音轉文字
- 匹配背景音樂和音效
- 影片格式調整和優化

### 👤 你負責的工作（20%）
- 定義影片目標和受眾
- 撰寫或提供影片主題和核心內容
- 選擇影片風格和調性（正式/輕鬆/專業）
- 審閱和微調 AI 生成的影片內容
- 確認影片長度和格式適合平台
- 最終審核和發布

### 📝 實作步驟
1. **定義影片需求**：
   - 告訴 ChatGPT 或 Claude 你的影片目標、受眾、主題、長度（建議 15-60 秒）
   - 例如：「我需要製作一個 30 秒的產品介紹影片，目標受眾是年輕族群，風格輕鬆活潑」

2. **AI 生成腳本和大綱**：
   - 請 AI 根據需求生成影片腳本，包含：
     - 開場（吸引注意力）
     - 核心內容（主要訊息）
     - 結尾（行動呼籲）
   - 腳本包含畫面描述、文字內容、字幕建議

3. **選擇影片生成工具**（根據需求選擇）：

   **方法 A：使用 Gemini Veo 3（簡單快速）**
   - 在 Gemini 中開啟影片生成功能
   - 輸入文字 prompt，例如：「一個年輕人在咖啡店裡使用我們的產品，環境輕鬆愉快，背景音樂輕快」
   - Veo 會生成約 8 秒的短影片
   - **適用**：快速生成短片片段、視覺效果

   **方法 B：使用 Invideo AI 或 Simplified（完整影片）**
   - 登入 Invideo AI 或 Simplified 免費帳號
   - 選擇「文字轉影片」或「腳本轉影片」功能
   - 貼上 AI 生成的腳本
   - 選擇模板風格（教學/商業/社群等）
   - AI 自動生成包含字幕、語音、素材的完整影片

   **方法 C：使用 VEED 或 Kapwing（編輯優化）**
   - 上傳圖片、影片素材或使用文字生成
   - 使用 AI 功能：自動生成字幕、語音轉文字、背景音樂建議
   - 手動調整字幕位置、顏色、字體
   - 添加轉場效果和視覺元素

4. **整合多個工具（進階）**：
   - 使用 **ChatGPT/Claude** 生成腳本和概念
   - 使用 **Gemini Veo** 或 **Simplified** 生成主要影片內容
   - 使用 **Canva** 製作封面圖或視覺元素
   - 使用 **VEED** 進行最終編輯和字幕調整

5. **影片優化和調整**：
   - 檢查字幕是否正確、同步
   - 確認音量平衡（語音和背景音樂）
   - 調整影片格式（9:16 適合手機社群，16:9 適合簡報）
   - 移除或接受浮水印（免費版通常有浮水印）

6. **輸出和分享**：
   - 匯出影片（MP4 格式，建議 720p 或 1080p）
   - 用於簡報、社群媒體、教學平台等

### 📄 產出結果
- **完成的短影片**（15-60 秒，包含字幕和音軌）
- **影片腳本文件**（可重複使用和修改）
- **視覺素材庫**（AI 選擇的素材可保存）

### 💡 實際應用範例

**範例 1：教學說明短片**
- **需求**：解釋一個數學概念（如「什麼是微積分」）
- **流程**：
  1. 請 ChatGPT 撰寫 30 秒的教學腳本
  2. 使用 **Simplified** 或 **Lumen5** 將腳本轉成影片
  3. AI 自動選擇數學相關的視覺素材和動畫
  4. 自動生成語音和字幕
- **產出**：30 秒教學短片，可在課堂使用或上傳 YouTube Shorts

**範例 2：產品介紹影片**
- **需求**：為新產品製作 45 秒的宣傳影片
- **流程**：
  1. 請 Claude 生成產品介紹腳本（開場、特色、結尾）
  2. 使用 **Invideo AI** 選擇商業模板
  3. 上傳產品圖片，AI 自動生成動畫效果
  4. 加入品牌音樂和字幕
- **產出**：45 秒產品宣傳影片，可用於社群媒體

**範例 3：簡報動畫短片**
- **需求**：製作簡報開場的動畫短片（15 秒）
- **流程**：
  1. 使用 **Gemini Veo 3** 從文字 prompt 生成短動畫
  2. 或在 **VEED** 中結合文字、圖片、動畫效果
  3. 添加背景音樂和轉場效果
- **產出**：15 秒動畫短片，嵌入簡報中

**範例 4：社群貼文影片（Reels/Short）**
- **需求**：製作 Instagram Reels 或 TikTok 影片（60 秒）
- **流程**：
  1. 請 ChatGPT 撰寫社群風格腳本（輕鬆、有趣）
  2. 使用 **Simplified** 或 **VEED** 製作 9:16 格式影片
  3. 選擇流行音樂和視覺效果
  4. 自動生成字幕（社群平台通常需要字幕）
- **產出**：60 秒社群影片，符合平台格式要求

- **教學平台**：16:9 或 9:16，30-60 秒，1080p
