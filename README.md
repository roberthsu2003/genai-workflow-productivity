# 生成式AI全面探索:技術、應用與未來趨勢

## 目標

✔ 把 AI 當成「每天一起工作的同事」

✔ 用 AI **減少重複、瑣碎、低價值工作**

✔ 知道如何把 AI **嵌入既有流程，而不是額外負擔**

✔ 具備 AI 素養，不被新工具牽著走

### **主題：生成式 AI 全貌與企業應用地圖**

#### **1. 生成式 AI 是什麼**

- 從 ChatGPT 到企業 AI
- LLM、Diffusion、Multimodal 的角色
- AI 能做什麼？不能做什麼？
- **AI 幻覺與風險意識**

#### **2. 企業為什麼需要生成式 AI**

- 個人生產力 vs 團隊生產力
- AI 對不同部門的影響
    - 行政、人資、行銷、業務、IT、主管
- 成功案例解析（流程優化，不談炒作）

#### **3. 主流生成式 AI 工具總覽**

- ChatGPT / Claude / Gemini
- Copilot 類工具（Office / Coding）
- 圖像、文件、簡報生成
- **雲端 AI vs 本地端 AI（資料安全）**

#### **4. 實作：AI 作為你的數位助理**

- 工作拆解：每天實際在做什麼？
- 將「人類工作」轉為「AI 可協助的任務」
- 建立第一份 **個人 AI 工作清單**

### **主題：Prompt Engineering × 工作場景**

#### **1. Prompt Engineering 核心觀念**

- 為什麼 Prompt ≠ 問問題
- 結構化 Prompt（角色、任務、限制、輸出）
- 常見錯誤示範

#### **2. 各職能 Prompt 範本**

- 行政 / 人資 / 行銷 / 業務 / 技術
- 會議摘要、郵件、企劃、報告
- Prompt 模板化思維

#### **3. 實作：建立公司 Prompt Library**

- 建立部門專屬 Prompt
- Prompt 版本管理概念
- 將 Prompt 存入 repository

### **主題：AI × 文件 × 知識管理（RAG）**

#### **1. AI 為什麼「不懂公司內部資料」**

- LLM 的知識限制
- 企業資料的價值

#### **2. RAG（檢索增強生成）概念**

- 向量資料庫是什麼
- RAG 在企業的應用場景
- 不談數學，談「用途」

#### **3. 實作：公司文件 AI 助理**

- 將 PDF / Word / SOP 丟給 AI
- 建立「問公司文件的 AI」
- 比較：沒有 RAG vs 有 RAG

### **主題：AI Workflow × 自動化**

#### **1. 什麼是 Workflow 思維**

- 人類流程 vs AI 流程
- Trigger → AI → Action

#### **2. AI + 自動化工具**

- n8n / Zapier / Make 概念
- AI 在流程中的角色
- 實務案例解析

#### **3. 實作：AI 自動化工作流**

- 表單 → AI 分析 → 文件產出
- Email → AI 分類 → 回覆建議
- 每組完成一條可運作流程

### **主題：AI Agent 與進階應用**

#### **1. 什麼是 AI Agent**

- Agent vs Chatbot
- 多步驟任務的自動執行

#### **2. Agent 在企業的應用**

- 自動報告生成
- 資料蒐集與分析
- 決策輔助

#### **3. 實作：打造簡單 AI Agent**

- 拆解任務 → Agent 流程
- Agent + Workflow 結合
- 示範「不寫或少寫程式」

### **主題：AI 導入策略、風險與未來**

#### **1. AI 資安、隱私與合規**

- 公司資料可以丟給 AI 嗎？
- 內部規範怎麼訂
- 本地端模型 vs 雲端模型

#### **2. AI 導入企業的實務策略**

- 從「個人使用」到「組織導入」
- AI Champion 制度
- KPI 與效益衡量

#### **3. 生成式 AI 未來趨勢**

- Multimodal
- Agentic Workflow
- AI 與人類的協作模式

#### **4. 最終專題發表**

- 每組展示：
- 一個 AI 工作流
- 對應實際工作場景
- 講師回饋與優化建議
