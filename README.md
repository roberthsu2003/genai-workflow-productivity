### ⚙️ MCP 設定說明（免程式）

**本地檔案系統 MCP**（免費，預設可用）
- 在 Claude Desktop 設定中，啟用「Filesystem」MCP
- 指定可存取的資料夾路徑
- 完成後，Claude 可以直接讀取這些資料夾的檔案

**Google Drive MCP**（免費方案可用）
- 在設定中選擇「Google Drive」Connector
- 點選授權按鈕，登入 Google 帳號
- 授予 Claude 存取特定資料夾的權限
- 完成後，Claude 可以讀取你的 Google Drive 文件

**Notion MCP**（免費方案可用）
- 在 Notion 中建立 Integration Token
- 在 Claude Desktop 設定中輸入 Token
- 完成後，Claude 可以讀取指定的 Notion 頁面

**Canva MCP**（免費方案可用）
- 在 Claude Desktop 設定中選擇「Canva」Connector
- 點選授權按鈕，登入 Canva 帳號
- 授予 Claude 存取權限
- 完成後，Claude 可以直接在 Canva 中生成和編輯設計
- **使用範例**：告訴 Claude「幫我在 Canva 製作一張關於本週銷售數據的資訊圖表」，Claude 會自動在 Canva 中建立設計

### ⚙️ Claude Skills 設定說明（免程式）

**內建 Skills**（免費方案可用）
- Claude Desktop 已預設包含多種內建 Skills
- 如：Excel 處理、PowerPoint 生成、PDF 分析、文字格式化等
- 直接使用相關指令即可觸發，例如：「請分析這個 Excel 檔案」
- **不需額外設定**，直接可用

**自訂 Skills**（進階方案功能更完整）
- 在 Claude Desktop 設定 → Capabilities → Skills 中啟用
- 可以建立任務專屬的資料夾，包含：
  - 角色說明（例如：「你是專業的報告撰寫助手」）
  - 標準格式範本（例如：公司報告格式）
  - 處理規則（例如：特定術語的寫法）
- **使用方式**：在任務描述中，Claude 會自動套用相關 Skills
- **注意**：完整自訂 Skills 功能可能需要 Pro/Max/Team 方案

**MCP + Skills 結合使用**
- **MCP** 負責：連接外部資料來源（Google Drive、Notion、本地檔案等）
- **Skills** 負責：套用處理規則和格式範本
- **範例**：「從 Google Drive 抓取報告範本（MCP），依公司格式（Skills）填入本週資料」

### 💡 免費方案使用技巧
- ⚠️ **注意權限**：授權時只給予必要的存取權限，保護資料安全
- ✅ **本地優先**：敏感資料使用本地檔案系統 MCP，不上傳雲端
- 🔄 **分批處理**：大量資料可分多次處理，避免一次處理過多導致延遲
- 📦 **組合使用**：結合多個免費 Connector，擴大資料整合範圍

### 🔒 資料安全提醒
- ⚠️ **敏感資料**：機密文件建議只使用本地檔案系統 MCP
- ✅ **一般資料**：可使用 Google Drive、Notion 等雲端 Connector
- 🔐 **權限控制**：定期檢查已授權的應用程式，移除不需要的存取權限

| **Simplified** | 文字/文章→影片 | 有限額度、基本功能 | 文章轉影片、教學影片 |
| **VEED** | 影片編輯、字幕生成 | 基本功能、有浮水印 | 影片編輯優化 |
| **Lumen5** | 文章/腳本→影片 | 免費版功能有限 | 內容行銷影片 |

### 💡 免費方案使用技巧
- ⚠️ **浮水印處理**：免費版通常有浮水印，若不影響使用可接受，或考慮付費去除
- ✅ **分散使用**：不同工具用於不同場景，避免單一工具額度耗盡
- 🔄 **重複使用腳本**：將 AI 生成的腳本保存，可重複用於不同工具
- 📦 **組合使用**：用 ChatGPT 生成腳本 + Simplified 製作影片 + VEED 編輯優化

### 🎬 影片格式建議
- **社群媒體**：9:16（直式），15-60 秒，720p 或 1080p
- **簡報嵌入**：16:9（橫式），15-30 秒，1080p
- **教學平台**：16:9 或 9:16，30-60 秒，1080p

### 🔒 資料安全提醒
- ⚠️ **版權注意**：確認使用的音樂、圖片素材是否有版權限制
- ✅ **公開使用**：大部分 AI 生成的影片內容可公開使用，但需檢查工具的使用條款
- 🔐 **個人資料**：避免在影片中包含敏感個人資訊
