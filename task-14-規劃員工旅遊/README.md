# 任務 14：規劃員工旅遊

## 🎯 工作情境
需要為公司規劃員工旅遊行程，但需要比較多個目的地、景點、交通和住宿選項，手動搜尋和比價非常耗時。例如：國內或國外旅遊、單日或多日行程、不同預算方案等。

## 🤖 使用的 AI 工具
- **Perplexity**（AI 搜尋引擎，免費版可用，Pro 版功能更完整）
- **Claude** 或 **ChatGPT**（整理行程提案）
- **Google Sheets** 或 **Excel**（整理行程和比價表格）
- **Google Docs**（行程提案文件）

## 💻 AI 負責的工作（80%）
- 自動搜尋旅遊目的地推薦和景點資訊
- 比較多個來源的交通方式、費用和時間
- 搜尋住宿選項並比價（飯店、民宿、度假村等）
- 整合天氣、季節、旅遊提示等資訊
- 生成行程時間表和路線規劃
- 提供多方案比較（高/中/低預算）
- 整理資料來源引用，方便後續查證

## 👤 你負責的工作（20%）
- 定義旅遊基本條件（預算、人數、天數、目的地偏好）
- 決定旅遊風格和活動偏好（自然、美食、文化、休閒等）
- 審閱 AI 搜尋和整理的資訊準確性
- 選擇最終方案和確認細節
- 製作最終行程提案並分享給主管或同事

## 📝 實作步驟
1. **定義旅遊需求**：確定預算、人數、天數、目的地偏好和旅遊風格
2. **AI 搜尋資訊**：使用 Perplexity 搜尋目的地、景點、交通、住宿等資訊
3. **資料整理**：將搜尋結果整理成表格（行程表、住宿比價表、預算比較表）
4. **生成行程提案**：使用 Claude/ChatGPT 整理成完整的行程提案文件
5. **審閱調整**：檢查資訊準確性、調整行程安排、確認預算
6. **製作最終提案**：整理成 Google Docs，加入視覺元素，分享給團隊

## 📄 產出結果
- 完整的行程提案文件（包含每日行程、時間表、預算明細）
- 多方案比較表（高/中/低預算方案）
- 住宿和交通比價表（包含價格、評價、優缺點）
- 資料來源引用清單（方便後續查證和確認）
