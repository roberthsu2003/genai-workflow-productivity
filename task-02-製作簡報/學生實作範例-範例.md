# 任務 2：製作簡報（學生實作範例）

> **本範例示範學生如何完成簡報製作作業**

---

## 📋 簡報基本資訊

- **簡報主題**：校園環保行動計畫提案
- **簡報目標**：向學校行政單位提案，爭取支持校園環保活動
- **目標受眾**：學校行政人員、老師、學生代表
- **簡報長度**：15 頁
- **簡報時間**：10 分鐘
- **製作時間**：3 天
- **製作人**：王同學（使用 AI 工具協助）

## 🎯 簡報大綱生成

### 步驟 1：定義需求

**問題回答：**

1. 簡報的主要目的是什麼？
   - 向學校行政單位提案，爭取支持校園環保活動
   - 說明活動的意義和執行方式
   - 獲得預算和行政支援

2. 目標受眾是誰？他們最關心什麼？
   - 學校行政人員：關心活動的可行性、成本、成效
   - 老師：關心如何融入教學、對學生的影響
   - 學生代表：關心活動的趣味性和參與度

3. 核心訊息是什麼？
   - 透過校園環保行動，培養學生的環保意識，並實際改善校園環境

4. 需要包含哪些主要部分？
   - ✅ 活動背景與目標
   - ✅ 現況分析
   - ✅ 活動規劃
   - ✅ 預算與資源需求
   - ✅ 預期成效

### 步驟 2：生成大綱提示詞

**使用的提示詞：**

```
請為我生成一份 15 頁簡報的大綱，主題是「校園環保行動計畫提案」。
目標受眾是學校行政人員和老師，需要包含：
1. 活動背景與目標
2. 現況分析（校園環保現狀、問題點）
3. 活動規劃（具體活動內容、執行方式）
4. 預算與資源需求
5. 預期成效

請提供每頁的標題和內容重點。
```

## 📝 內容生成記錄

### 頁面 1：為什麼需要環保行動？

**使用的 AI 工具：** ChatGPT

**提示詞：**
```
請為我的簡報第 3 頁撰寫內容，主題是「為什麼需要環保行動？」。
這是一個向學校行政單位提案的簡報。
請包含：
1. 環保的重要性
2. 校園環保的意義
3. 學生的責任
請以簡報風格撰寫，每段約 3-5 句話。
```

**AI 生成內容：**
```
為什麼需要環保行動？

環保是當今全球最重要的議題之一：
- 氣候變遷、資源耗竭、環境污染等問題日益嚴重
- 每個人都有責任為地球的永續發展盡一份力
- 校園是培養環保意識的最佳場所

校園環保的意義：
- 從小培養學生的環保習慣，影響其一生
- 實際改善校園環境，創造更美好的學習空間
- 建立環保典範，影響家庭和社區

學生的責任：
- 作為未來的主人翁，我們有責任保護環境
- 透過實際行動，展現年輕世代的責任感
- 將環保理念落實於日常生活中
```

**我的調整：**
- 加入了一些實際的數據和例子
- 調整了語氣，讓它更適合學校行政人員閱讀
- 增加了視覺化建議（例如：使用地球的圖片）

### 頁面 2：校園環保現狀調查

**使用的 AI 工具：** ChatGPT + Perplexity

**提示詞：**
```
請為我的簡報第 6 頁撰寫內容，主題是「校園環保現狀調查」。
請包含：
1. 問卷調查結果
2. 實際觀察到的問題
3. 數據分析
請以簡報風格撰寫，使用數據和圖表建議。
```

**AI 生成內容：**
```
校園環保現狀調查

問卷調查結果（回收 300 份有效問卷）：
- 78% 的學生認為校園環保很重要
- 但只有 45% 的學生會主動進行資源回收
- 62% 的學生不清楚垃圾分類的具體規則

實際觀察到的問題：
- 垃圾桶內混雜可回收和不可回收垃圾
- 塑膠垃圾比例過高，一次性餐具使用普遍
- 環保觀念與實際行為有落差

數據分析：
- 校園每日產生的垃圾量：約 150 公斤
- 可回收垃圾比例：約 35%（可提升至 60%）
- 塑膠垃圾比例：約 40%（目標降至 20%）
```

**我的調整：**
- 實際進行了問卷調查，替換了 AI 生成的假數據
- 使用 Perplexity 查詢了其他學校的環保數據作為參考
- 加入實際拍攝的校園照片作為證據

### 頁面 3：活動規劃 - 廢棄物減量計畫

**使用的 AI 工具：** ChatGPT

**提示詞：**
```
請為我的簡報第 10 頁撰寫內容，主題是「廢棄物減量計畫」。
請包含具體的活動內容：
1. 減塑行動
2. 零廢棄日
3. 環保餐具推廣
請說明每個活動的執行方式和預期效果。
```

**AI 生成內容：**
```
廢棄物減量計畫

減塑行動：
- 推廣自備環保杯、環保袋
- 校園商店提供環保餐具租借服務
- 舉辦減塑挑戰賽，鼓勵學生參與

零廢棄日：
- 每個月選一天為「零廢棄日」
- 當天禁止使用一次性餐具和塑膠製品
- 宣傳環保理念，提升環保意識

環保餐具推廣：
- 與校園商家合作，提供環保餐具優惠
- 設立環保餐具租借站
- 製作宣傳海報和影片
```

**我的調整：**
- 調整了活動的具體執行細節，使其更符合學校的實際情況
- 加入了與校園商家的合作計畫（實際已聯繫）
- 增加了活動的時程規劃

## 💡 製作過程中的學習與反思

### 遇到的最大挑戰

**挑戰：** 如何平衡 AI 生成的通用內容和實際的校園情況

**描述：** AI 生成的內容往往比較通用，例如預算規劃可能不符合學校的實際情況。需要花時間調整內容，使其更貼近實際。

### 解決方法

1. **實際調研**：進行問卷調查和實際觀察，獲得真實數據
2. **多方驗證**：查詢多個來源，確認數據的準確性
3. **反覆調整**：根據實際情況多次調整 AI 生成的內容

### 如果重來一次，你會如何改進

1. **提前準備數據**：在開始製作前就先進行調研，準備好真實數據
2. **建立資料庫**：建立一個資料庫，收集相關的圖片、數據和案例
3. **分階段完成**：將工作分成更小的階段，每階段完成後再進行下一階段

### 建議與心得

1. **AI 是輔助工具**：AI 可以大幅提升效率，但不能完全依賴
2. **人工判斷重要**：需要人工判斷和調整，確保內容符合實際需求
3. **數據驗證必要**：所有數據都需要驗證，不能直接使用
4. **反覆優化**：完成初稿後要多次檢視和調整，確保品質
