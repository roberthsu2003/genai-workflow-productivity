# 任務 2：製作簡報（從零到完成）

## 🎯 工作情境
需要在 3 天內完成一份 20 頁的產品提案簡報，包含市場分析、產品介紹、財務預測。

## 🤖 使用的 AI 工具
- **ChatGPT** 或 **Claude**（內容大綱生成）
- **Google Slides**（簡報製作，內建 AI 輔助）
- **Canva**（視覺化簡報設計，Magic Design 生成模板）
- **Perplexity**（市場數據研究）
- **Gemini**（多模態內容整合）

## 💻 AI 負責的工作（80%）
- 根據主題生成簡報大綱
- 撰寫各頁文字內容
- 生成圖表數據建議
- 設計簡報結構和流程
- 提供視覺化建議

## 👤 你負責的工作（20%）
- 定義簡報目標和受眾
- 提供產品核心資訊
- 確認數據來源的準確性
- 調整簡報風格符合品牌
- 最終簡報前的檢查和潤飾

## 📝 實作步驟
1. **定義需求**：告訴 ChatGPT 你的簡報目標、受眾、核心訊息
2. **生成大綱**：AI 產生 20 頁的簡報架構
3. **內容生成**：逐頁請 AI 撰寫內容，並用 Perplexity 查詢市場數據
4. **簡報製作**：選擇 Google Slides 或 Canva 進行製作
5. **視覺優化**：根據 AI 建議製作圖表和視覺元素
6. **最終調整**：檢查邏輯流程、修正品牌色調、確認數據準確性

## 📄 產出結果
- 完整的 20 頁簡報（Google Slides 或 Canva）
- 簡報大綱文件（可作為會議討論基礎）
- 數據來源清單（便於後續更新）
