# 任務 4：郵件撰寫與回覆

## 🎯 工作情境
每天需要回覆大量客戶詢問郵件，內容包含產品諮詢、技術支援、客訴處理等不同類型。

## 🤖 使用的 AI 工具
- **Gmail 智慧撰寫**（內建 AI 建議）
- **ChatGPT** 或 **Claude**（郵件草稿生成）
- **Gemini**（多郵件批量處理）

## 💻 AI 負責的工作（80%）
- 根據郵件內容自動分類（詢問/客訴/感謝）
- 生成回覆草稿
- 調整語氣（正式/友善/專業）
- 翻譯多語言郵件
- 提取客戶郵件的關鍵資訊

## 👤 你負責的工作（20%）
- 確認郵件分類是否正確
- 檢查 AI 回覆是否符合公司政策
- 加入個人化內容（客戶名稱、具體案例）
- 決定回覆的優先順序
- 最終確認後發送

## 📝 實作步驟
1. **郵件分類**：將客戶郵件貼給 ChatGPT，要求分類並提取關鍵資訊
2. **生成回覆**：請 AI 根據郵件類型和公司模板生成回覆草稿
3. **調整語氣**：指定語氣（正式/親切），AI 調整用詞
4. **個人化**：手動加入客戶名稱和具體細節
5. **利用 Gmail 智慧撰寫**：在 Gmail 中撰寫時，AI 會自動建議完整句子
6. **批量處理**：將多封相似郵件一併處理，使用模板快速回覆

## 📄 產出結果
- 分類好的郵件清單（優先順序）
- 標準回覆模板庫（可重複使用）
- 已回覆的郵件記錄（追蹤用）
