# 任務 4：郵件撰寫與回覆（老師範例）

> **本範例示範如何使用 AI 工具（Gmail 智慧撰寫、ChatGPT/Claude、Gemini）協助撰寫與回覆郵件**

---

## 📋 工作情境

- **工作內容**：每天需要回覆大量客戶詢問郵件
- **郵件類型**：產品諮詢、技術支援、客訴處理、感謝函
- **處理數量**：每天約 30-50 封郵件
- **使用時間**：原本需要 4 小時，使用 AI 後縮短至 1 小時
- **處理人**：客服專員（使用 AI 工具協助）

---

## 📧 郵件分類與分析

### 範例郵件 1：產品諮詢

**原始客戶郵件：**

```
主題：想了解你們的產品A功能

您好，
我對你們的產品A很感興趣，想了解一下：
1. 產品A有哪些主要功能？
2. 適合什麼樣的使用者？
3. 價格是多少？
4. 有試用版嗎？

期待您的回覆。

王先生
聯絡電話：0912-345-678
```

**步驟 1：使用 ChatGPT 分類與提取資訊**

**提示詞：**
```
請幫我分析以下客戶郵件：
1. 郵件類型（產品諮詢/技術支援/客訴/其他）
2. 提取關鍵資訊（客戶需求、聯絡方式等）
3. 客戶的緊急程度（高/中/低）
4. 建議回覆重點

[貼上客戶郵件]
```

**AI 分析結果：**

- **郵件類型**：產品諮詢
- **關鍵資訊**：
  - 產品名稱：產品A
  - 詢問項目：功能、適用對象、價格、試用版
  - 聯絡方式：王先生，0912-345-678
- **緊急程度**：中（一般詢問）
- **建議回覆重點**：
  - 詳細介紹產品A功能
  - 說明適用對象
  - 提供價格資訊
  - 說明試用版取得方式


## 📧 範例郵件 2：技術支援

**原始客戶郵件：**

```
主題：產品A無法正常匯入Excel檔案

您好，
我使用產品A時遇到問題：
- 無法匯入Excel檔案（.xlsx格式）
- 出現錯誤訊息：「檔案格式不支援」
- 我使用的Excel版本是2021

請問該如何解決？

謝謝！
李小姐
```

**AI 分類與分析：**

- **郵件類型**：技術支援
- **緊急程度**：高（影響使用）
- **問題描述**：Excel檔案匯入失敗
- **技術細節**：.xlsx格式、Excel 2021

**使用 ChatGPT 生成回覆：**

**提示詞：**
```
請撰寫一封技術支援回覆郵件：

客戶問題：無法匯入Excel檔案
錯誤訊息：「檔案格式不支援」
Excel版本：2021

解決方案：
1. 檢查檔案是否損壞
2. 確認檔案大小（限制10MB）
3. 嘗試另存為.xls格式
4. 聯絡技術支援（如果仍無法解決）

語氣：專業、友善、提供具體步驟
```

**AI 生成的回覆：**

```
主題：Re: 產品A無法正常匯入Excel檔案

李小姐您好，

感謝您聯絡我們！關於您遇到的Excel檔案匯入問題，我提供以下解決方案：

**可能原因與解決方法：**

1. **檢查檔案完整性**
   - 請確認Excel檔案可以正常開啟
   - 建議先在其他電腦測試開啟

2. **確認檔案大小**
   - 產品A支援的檔案大小上限為10MB
   - 如果檔案過大，請考慮分割或壓縮

3. **嘗試轉換格式**
   - 請在Excel中「另存新檔」
   - 選擇「Excel 97-2003活頁簿(.xls)」格式
   - 重新嘗試匯入

4. **如果以上方法都無法解決**
   - 請將錯誤截圖和檔案範例寄給我們
   - 我們會由技術團隊協助處理
   - 聯絡方式：tech@example.com

如果問題持續，或您需要即時協助，歡迎來電：02-1234-5678

期待能盡快協助您解決問題！

[您的姓名]
技術支援專員
```

## 📧 範例郵件 3：客訴處理

**原始客戶郵件：**

```
主題：對產品A的服務非常不滿

您好，
我對你們的服務非常失望：
1. 產品功能與宣傳不符

3. 取消訂閱流程複雜

我要求退款，請盡快處理。

張先生
```

**AI 分析與處理策略：**

- **郵件類型**：客訴
- **緊急程度**：高（涉及退款）
- **情緒狀態**：不滿、憤怒
- **處理策略**：
  1. 立即道歉
  2. 理解客戶感受
  3. 提供解決方案
  4. 說明改進措施

**使用 ChatGPT 生成客訴回覆：**

**提示詞：**
```
請撰寫一封客訴處理郵件：

客戶問題：
1. 產品功能與宣傳不符
2. 客服回應慢
3. 取消訂閱流程複雜
4. 要求退款

要求：
1. 語氣：誠懇、道歉、積極解決
2. 立即處理退款申請
3. 提供改進承諾
4. 邀請進一步溝通

語氣要非常友善和專業，展現解決問題的誠意。
```

**AI 生成的回覆：**

## 💡 製作技巧與注意事項

### 郵件撰寫最佳實踐

1. **明確郵件目的**
   - 確認郵件類型和目標
   - 這將影響語氣和結構

2. **個人化內容**
   - 使用客戶姓名
   - 提及具體的問題或需求
   - 展現對客戶的重視

3. **清晰結構**
   - 感謝→回答問題→提供額外資訊→行動呼籲
   - 使用項目符號和分段
   - 重點資訊加粗

4. **語氣調整**
   - 根據郵件類型調整語氣
   - 客訴郵件要更誠懇
   - 技術支援要更專業

5. **行動呼籲**
   - 明確告知下一步行動
   - 提供聯絡方式
   - 表達願意進一步協助

### 常見錯誤避免

❌ **錯誤 1：過度依賴 AI**
- 問題：完全使用 AI 生成內容，未個人化
- 解決：AI 生成後要加入個人化內容

❌ **錯誤 2：語氣不當**
- 問題：所有郵件使用相同語氣
- 解決：根據郵件類型和客戶調整語氣

❌ **錯誤 3：缺乏個人化**
- 問題：使用模板但未替換客戶姓名
- 解決：每次都要確認個人化內容

❌ **錯誤 4：忽略客戶情緒**
- 問題：客訴郵件語氣不夠誠懇
- 解決：根據客戶情緒調整語氣和內容
