# 任務 4：郵件撰寫與回覆（學生實作範例）

> **本範例示範學生如何完成郵件撰寫與回覆作業**

---

## 📋 郵件處理資訊

- **處理日期**：2025年1月15日
- **處理郵件數量**：5 封
- **郵件類型**：產品諮詢 3 封、技術支援 1 封、客訴 1 封
- **處理時間**：30 分鐘（原本需要 2 小時）
- **處理人**：陳同學（使用 AI 工具協助）

---

## 📧 郵件 1：產品諮詢

### 原始客戶郵件

```
主題：想了解課程內容
內容：
您好，
我對你們的線上課程很有興趣，想了解一下：
1. 課程內容包含哪些？
2. 上課方式是？
3. 費用是多少？
4. 有試聽嗎？

期待您的回覆。

王同學
```

### 步驟 1：使用 ChatGPT 分類與提取資訊

**提示詞：**
```
請幫我分析以下客戶郵件：
1. 郵件類型
2. 提取關鍵資訊
3. 建議回覆重點

[貼上郵件內容]
```

**AI 分析結果：**

- **郵件類型**：產品諮詢
- **關鍵資訊**：
  - 詢問項目：課程內容、上課方式、費用、試聽
  - 客戶姓名：王同學
- **緊急程度**：中
- **建議回覆重點**：詳細回答所有問題

### 步驟 2：使用 ChatGPT 生成回覆草稿

**提示詞：**
```
請撰寫一封專業且友善的客戶回覆郵件：
[貼上郵件內容]

課程資訊：
- 課程內容：AI工具應用、簡報製作、數據分析
- 上課方式：線上錄播 + 定期直播
- 費用：3,990 元
- 試聽：提供第一單元免費試聽

語氣：友善、專業
```

**AI 生成的回覆：**

```
王同學您好，

感謝您對我們線上課程的興趣！

以下是您詢問的問題回覆：

1. 課程內容
   我們的課程包含三大主題：
   - AI工具應用：學習使用ChatGPT、Claude等工具
   - 簡報製作：使用AI輔助製作專業簡報
   - 數據分析：用AI工具進行數據分析與視覺化

2. 上課方式
   - 主要為線上錄播課程，可隨時學習
   - 每週有定期直播，可與講師互動
   - 提供學習群組，可與同學交流

3. 費用
   - 完整課程：3,990 元
   - 早鳥優惠：3,590 元（限時）

4. 試聽
   我們提供第一單元免費試聽，不需註冊即可觀看。
   連結：https://example.com/trial

如有其他問題，歡迎隨時聯繫！

祝學習愉快！

陳同學
課程助理
```

### 步驟 3：個人化調整

**我的調整：**
- ✅ 確認客戶姓名正確
- ✅ 檢查所有資訊完整
- ✅ 語氣友善且專業
- ✅ 加入聯絡方式

## 🤖 AI 工具使用總結

**使用的工具：** ChatGPT、Gmail 智慧撰寫

**優點：**
- AI 能快速分類郵件
- 生成回覆草稿迅速
- 語氣調整靈活

**缺點：**
- 需要人工確認內容
- 個人化內容需手動加入
